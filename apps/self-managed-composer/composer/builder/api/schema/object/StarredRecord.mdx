## Return fields

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the StarredRecord.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the StarredRecord.

###### <b>description</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>timestamp</b> ([DateTime!](/composer/builder/api/schema/scalar/DateTime.mdx))

Date/Time when the record was starred

###### <b>user</b> ([User!](/composer/builder/api/schema/object/User.mdx))

User that starred a record
