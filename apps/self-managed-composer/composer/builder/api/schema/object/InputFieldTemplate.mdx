## Return fields

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the InputFieldTemplate.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the InputFieldTemplate.

###### <b>connectionTemplate</b> ([ConnectionTemplate!](/composer/builder/api/schema/object/ConnectionTemplate.mdx))

The template that this input is associated with.

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>inputField</b> ([InputField!](/composer/builder/api/schema/object/InputField.mdx))

The InputField that this template is associated with.

###### <b>value</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

The preset value for this input.
