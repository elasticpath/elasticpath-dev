Represents a specific value of a CredentialField.

## Return fields

###### <b>key</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

The name associated with the CredentialField.

###### <b>value</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

The value of the CredentialField.
