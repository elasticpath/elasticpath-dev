Represents a Relay Connection to a collection of InstanceExecutionResult objects.

## Return fields

###### <b>edges</b> ([[InstanceExecutionResultEdge]!](/composer/builder/api/schema/object/InstanceExecutionResultEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[InstanceExecutionResult]!](/composer/builder/api/schema/object/InstanceExecutionResult.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](/composer/builder/api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](/composer/builder/api/schema/scalar/Int.mdx))

Total count of nodes available.
