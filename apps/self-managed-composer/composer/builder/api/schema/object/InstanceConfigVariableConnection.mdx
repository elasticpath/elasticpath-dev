Represents a Relay Connection to a collection of InstanceConfigVariable objects.

## Return fields

###### <b>edges</b> ([[InstanceConfigVariableEdge]!](/composer/builder/api/schema/object/InstanceConfigVariableEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[InstanceConfigVariable]!](/composer/builder/api/schema/object/InstanceConfigVariable.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](/composer/builder/api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](/composer/builder/api/schema/scalar/Int.mdx))

Total count of nodes available.
