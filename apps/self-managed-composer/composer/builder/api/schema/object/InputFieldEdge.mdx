A Relay edge to a related InputField object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([InputField](/composer/builder/api/schema/object/InputField.mdx))

The related object at the end of the edge.
