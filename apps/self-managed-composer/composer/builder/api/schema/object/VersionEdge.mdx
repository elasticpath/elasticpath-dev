A Relay edge containing a `Version` and its cursor.

## Return fields

###### <b>cursor</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

A cursor for use in pagination

###### <b>node</b> ([Version](/composer/builder/api/schema/object/Version.mdx))

The item at the end of the edge
