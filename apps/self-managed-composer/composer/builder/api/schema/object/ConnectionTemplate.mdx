## Return fields

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the ConnectionTemplate.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the ConnectionTemplate.

###### <b>connection</b> ([Connection!](/composer/builder/api/schema/object/Connection.mdx))

The Connection from which this template is structured.

###### <b>hasDeployedInstances</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Indicates template is in use on an Instance.

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>inputFieldTemplates</b> ([InputFieldTemplateConnection!](/composer/builder/api/schema/object/InputFieldTemplateConnection.mdx))

The template that this input is associated with.

###### <b>instances</b> ([InstanceConnection!](/composer/builder/api/schema/object/InstanceConnection.mdx))

Returns a list of deployed customer instances that are leveraging this template.

###### <b>name</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

The name of this template.

###### <b>templatedInputKeys</b> ([[String]](/composer/builder/api/schema/scalar/String.mdx))

Returns a list of the keys that are preset by this template.
