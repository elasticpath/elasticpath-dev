Provides dynamic user-driven config values to satisfy Required Config Variables of an Instance.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the UserLevelConfig.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the UserLevelConfig.

###### <b>configVariables</b> ([UserLevelConfigVariableConnection!](/composer/builder/api/schema/object/UserLevelConfigVariableConnection.mdx))

The Dynamic Config with which the Config Variable is associated.

###### <b>createdAt</b> ([DateTime!](/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was created.

###### <b>flowConfigs</b> ([UserLevelFlowConfigConnection!](/composer/builder/api/schema/object/UserLevelFlowConfigConnection.mdx))

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>instance</b> ([Instance!](/composer/builder/api/schema/object/Instance.mdx))

The Instance with which the User Level Config is associated.

###### <b>updatedAt</b> ([DateTime!](/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was most recently updated.

###### <b>user</b> ([User!](/composer/builder/api/schema/object/User.mdx))

The User that owns the User Level Config.
