The Relay compliant `PageInfo` type, containing data necessary to paginate this connection.

## Return fields

###### <b>endCursor</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

When paginating forwards, the cursor to continue.

###### <b>hasNextPage</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

When paginating forwards, are there more items?

###### <b>hasPreviousPage</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

When paginating backwards, are there more items?

###### <b>startCursor</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

When paginating backwards, the cursor to continue.
