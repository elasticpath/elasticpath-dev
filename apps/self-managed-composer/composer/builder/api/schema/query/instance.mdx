    Returns the specified Instance object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_admin_instance_permissions, org_manage_instances, org_view_instances]. 2. The signed-in User has any of the following permissions for the object: [instance_admin_permissions, instance_view, instance_edit, instance_remove]. 3. The signed-in User has any of the following permissions for the object's 'customer' attribute: [customer_admin_instance_permissions, customer_view_instances, customer_manage_marketplace_integrations]. 4. The signed-in User has any of the following permissions for the object's 'customer' attribute: [customer_access_marketplace_integrations]. 5. The signed-in User has any of the following permissions for the object's 'integration' attribute: [integration_admin_permissions, integration_view, integration_edit, integration_remove]. 6. The signed-in User has any of the following permissions for the object's 'integration_Customer' attribute: [customer_admin_manage_instances, customer_admin_integration_permissions, customer_manage_integrations].

## Input fields

| Argument | Type                              | Description                    |
| -------- | --------------------------------- | ------------------------------ |
| `id`     | [`ID!`](/composer/builder/api/schema/scalar/ID.mdx) | The ID of the Instance object. |

## Return fields ([Instance](/composer/builder/api/schema/object/Instance.mdx))

Specifies whether the signed-in User can deploy the Instance.

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Instance.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Instance.

###### <b>allowUpdateConfigVariables</b> ([Boolean](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update config variables for the Instance.

###### <b>configState</b> ([InstanceConfigState](/composer/builder/api/schema/enum/InstanceConfigState.mdx))

###### <b>configVariables</b> ([InstanceConfigVariableConnection!](/composer/builder/api/schema/object/InstanceConfigVariableConnection.mdx))

###### <b>createdAt</b> ([DateTime!](/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was created.

###### <b>customer</b> ([Customer!](/composer/builder/api/schema/object/Customer.mdx))

###### <b>deployedVersion</b> ([Int!](/composer/builder/api/schema/scalar/Int.mdx))

###### <b>description</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

###### <b>enabled</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

###### <b>executionResults</b> ([InstanceExecutionResultConnection!](/composer/builder/api/schema/object/InstanceExecutionResultConnection.mdx))

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

###### <b>inFailedState</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

###### <b>integration</b> ([Integration!](/composer/builder/api/schema/object/Integration.mdx))

###### <b>labels</b> ([[String!]](/composer/builder/api/schema/scalar/String.mdx))

###### <b>lastDeployedAt</b> ([DateTime](/composer/builder/api/schema/scalar/DateTime.mdx))

###### <b>lastExecutedAt</b> ([DateTime](/composer/builder/api/schema/scalar/DateTime.mdx))

###### <b>logs</b> ([LogConnection!](/composer/builder/api/schema/object/LogConnection.mdx))

###### <b>logsDisabled</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

###### <b>monitors</b> ([AlertMonitorConnection!](/composer/builder/api/schema/object/AlertMonitorConnection.mdx))

###### <b>name</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

###### <b>needsDeploy</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

###### <b>starred</b> ([Boolean](/composer/builder/api/schema/scalar/Boolean.mdx))

###### <b>stepResultsDisabled</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

###### <b>updatedAt</b> ([DateTime!](/composer/builder/api/schema/scalar/DateTime.mdx))
