    Returns the specified AlertWebhook object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_admin_instance_permissions, org_manage_instances, org_view_instances].

## Input fields

| Argument | Type                              | Description                        |
| -------- | --------------------------------- | ---------------------------------- |
| `id`     | [`ID!`](/composer/builder/api/schema/scalar/ID.mdx) | The ID of the AlertWebhook object. |

## Return fields ([AlertWebhook](/composer/builder/api/schema/object/AlertWebhook.mdx))

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the AlertWebhook.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the AlertWebhook.

###### <b>createdAt</b> ([DateTime!](/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was created.

###### <b>groups</b> ([AlertGroupConnection!](/composer/builder/api/schema/object/AlertGroupConnection.mdx))

The AlertWebhooks in the AlertGroup

###### <b>headers</b> ([JSONString](/composer/builder/api/schema/scalar/JSONString.mdx))

A JSON string of key/value pairs that will be sent as headers in the Webhook request.

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>monitors</b> ([AlertMonitorConnection!](/composer/builder/api/schema/object/AlertMonitorConnection.mdx))

The AlertWebhooks to call when the AlertMonitor is triggered.

###### <b>name</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

The name of the AlertWebhook.

###### <b>payloadTemplate</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

The template that is hydrated and then used as the body of the AlertWebhook request.

###### <b>updatedAt</b> ([DateTime!](/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was most recently updated.

###### <b>url</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

The URL of the AlertWebhook.
