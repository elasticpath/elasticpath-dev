    Returns the specified OrgTotalUsageMetrics object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_view_utilization].

## Input fields

| Argument | Type                              | Description                                |
| -------- | --------------------------------- | ------------------------------------------ |
| `id`     | [`ID!`](/composer/builder/api/schema/scalar/ID.mdx) | The ID of the OrgTotalUsageMetrics object. |

## Return fields ([OrgTotalUsageMetrics](/composer/builder/api/schema/object/OrgTotalUsageMetrics.mdx))

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the OrgTotalUsageMetrics.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the OrgTotalUsageMetrics.

###### <b>blobBytesStored</b> ([BigInt!](/composer/builder/api/schema/scalar/BigInt.mdx))

The total number of bytes of blob storage currently used.

###### <b>customerCount</b> ([Int!](/composer/builder/api/schema/scalar/Int.mdx))

The total number of Customers that currently exist.

###### <b>deployedInstanceCount</b> ([Int!](/composer/builder/api/schema/scalar/Int.mdx))

The total number of Instances that are deployed.

###### <b>deployedUniqueIntegrationCount</b> ([Int!](/composer/builder/api/schema/scalar/Int.mdx))

The total number of unique Integrations that are deployed.

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>integrationCount</b> ([Int!](/composer/builder/api/schema/scalar/Int.mdx))

The total number of Integrations that currently exist.

###### <b>snapshotTime</b> ([DateTime!](/composer/builder/api/schema/scalar/DateTime.mdx))

The time the utilization metrics snapshot was created.

###### <b>userCount</b> ([Int!](/composer/builder/api/schema/scalar/Int.mdx))

The total number of Users that currently exist.
