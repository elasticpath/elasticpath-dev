    Returns a Relay Connection to a collection of ConnectionTemplate objects.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_view]. 2. The signed-in User has any of the following permissions for the associated Customer: [customer_manage_integrations].

## Input fields

| Argument         | Type                                                                               | Description                                                                                      |
| ---------------- | ---------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------ |
| `connectionId`   | [`ID`](/composer/builder/api/schema/scalar/ID.mdx)                                                   | Connection associated with this template.                                                        |
| `before`         | [`String`](/composer/builder/api/schema/scalar/String.mdx)                                           | Specifies a cursor for use in combination with `last` to implement backward pagination.          |
| `after`          | [`String`](/composer/builder/api/schema/scalar/String.mdx)                                           | Specifies a cursor for use in combination with `first` to implement forward pagination.          |
| `first`          | [`Int`](/composer/builder/api/schema/scalar/Int.mdx)                                                 | A non-negative integer that specifies to return at most `first` edges after the `after` cursor.  |
| `last`           | [`Int`](/composer/builder/api/schema/scalar/Int.mdx)                                                 | A non-negative integer that specifies to return at most `last` edges before the `before` cursor. |
| `offset`         | [`Int`](/composer/builder/api/schema/scalar/Int.mdx)                                                 |                                                                                                  |
| `name_Icontains` | [`String`](/composer/builder/api/schema/scalar/String.mdx)                                           | Filter for objects where name contains the specified value (case insensitive).                   |
| `orderBy`        | [`ConnectionTemplateOrder`](/composer/builder/api/schema/input_object/ConnectionTemplateOrder.mdx)   | DEPRECATED. Prefer using sort_by instead as it allows ordering by many fields.                   |
| `sortBy`         | [`[ConnectionTemplateOrder]`](/composer/builder/api/schema/input_object/ConnectionTemplateOrder.mdx) | Allows specifying many field and direction pairs to sort results by.                             |

## Return fields ([ConnectionTemplateConnection!](/composer/builder/api/schema/object/ConnectionTemplateConnection.mdx))

###### <b>edges</b> ([[ConnectionTemplateEdge]!](/composer/builder/api/schema/object/ConnectionTemplateEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[ConnectionTemplate]!](/composer/builder/api/schema/object/ConnectionTemplate.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](/composer/builder/api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](/composer/builder/api/schema/scalar/Int.mdx))

Total count of nodes available.
