    Returns the specified Action object.

Access is permitted when any of the following condition(s) are met: 1. Always allowed.

## Input fields

| Argument | Type                              | Description                  |
| -------- | --------------------------------- | ---------------------------- |
| `id`     | [`ID!`](/composer/builder/api/schema/scalar/ID.mdx) | The ID of the Action object. |

## Return fields ([Action](/composer/builder/api/schema/object/Action.mdx))

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Action.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Action.

###### <b>allowsBranching</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Action will allow Conditional Branching.

###### <b>authorizationMethods</b> ([AuthorizationMethodConnection!](/composer/builder/api/schema/object/AuthorizationMethodConnection.mdx))

The AuthorizationMethods that are supported by the Action.

###### <b>authorizationRequired</b> ([Boolean](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Action requires authorization to function.

###### <b>breakLoop</b> ([Boolean](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether an Action will break out of a loop.

###### <b>component</b> ([Component](/composer/builder/api/schema/object/Component.mdx))

The Component to which this Action is associated.

###### <b>dataSourceType</b> ([ActionDataSourceType](/composer/builder/api/schema/enum/ActionDataSourceType.mdx))

Specifies the type of the resulting data from the data source.

###### <b>description</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

Additional notes about the Action.

###### <b>detailDataSource</b> ([Action](/composer/builder/api/schema/object/Action.mdx))

The Data Source in this Component which can provide additional details about the content for this Data Source, such as example values when selecting particular API object fields.

###### <b>directions</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

Notes which may provide insight on the intended use of the Action.

###### <b>dynamicBranchInput</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

A string that associates an Input with Dynamic Branching.

###### <b>examplePayload</b> ([JSONString](/composer/builder/api/schema/scalar/JSONString.mdx))

An example of the returned payload of an Action.

###### <b>hasOnInstanceDelete</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Action, if it is a Trigger, has an Instance Delete handler function defined.

###### <b>hasOnInstanceDeploy</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Action, if it is a Trigger, has an Instance Deploy handler function defined.

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>important</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Action is important and/or commonly used.

###### <b>inputs</b> ([InputFieldConnection!](/composer/builder/api/schema/object/InputFieldConnection.mdx))

The Action to which this InputField is associated, if any.

###### <b>isCommonTrigger</b> ([Boolean](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Action is a commonly used Trigger.

###### <b>isDataSource</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Action is a Data Source.

###### <b>isTrigger</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Action is a Trigger.

###### <b>key</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

A string that uniquely identifies this Action within the context of the Component.

###### <b>label</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

The name of the Action.

###### <b>scheduleSupport</b> ([ActionScheduleSupport](/composer/builder/api/schema/enum/ActionScheduleSupport.mdx))

Specifies support for triggering an Integration on a recurring schedule.

###### <b>searchTerms</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

A combination of an action's text metadata to be used in search functionality.

###### <b>staticBranchNames</b> ([[String!]](/composer/builder/api/schema/scalar/String.mdx))

The static branch names associated with an Action.

###### <b>synchronousResponseSupport</b> ([ActionSynchronousResponseSupport](/composer/builder/api/schema/enum/ActionSynchronousResponseSupport.mdx))

Specifies support for synchronous responses to an Integration webhook request.

###### <b>terminateExecution</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Action will terminate Instance execution.
