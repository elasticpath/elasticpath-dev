    Returns the specified AuthorizationMethod object.

Access is permitted when any of the following condition(s) are met: 1. Always allowed.

## Input fields

| Argument | Type                              | Description                               |
| -------- | --------------------------------- | ----------------------------------------- |
| `id`     | [`ID!`](/composer/builder/api/schema/scalar/ID.mdx) | The ID of the AuthorizationMethod object. |

## Return fields ([AuthorizationMethod](/composer/builder/api/schema/object/AuthorizationMethod.mdx))

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the AuthorizationMethod.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the AuthorizationMethod.

###### <b>description</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

Additional notes about the AuthorizationMethod.

###### <b>fields</b> ([CredentialFieldConnection!](/composer/builder/api/schema/object/CredentialFieldConnection.mdx))

The AuthorizationMethod that the CredentialField is associated to.

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>key</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

A string which uniquely identifies the AuthorizationMethod.

###### <b>label</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

The name of the AuthorizationMethod.
