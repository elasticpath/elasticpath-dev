    Returns a Relay Connection to a collection of User objects.

Access is permitted when any of the following condition(s) are met: 1. The specified object is the signed-in User. 2. The signed-in User has any of the following permissions for the associated Organization: [org_admin_users, org_view_users]. 3. The signed-in User has any of the following permissions for the associated Organization: [org_admin_customer_permissions, org_manage_customers, org_view_customers]. 4. The signed-in User has any of the following permissions for the object's 'customer' attribute: [customer_admin_users, customer_view_users].

## Input fields

| `createdAt_Lte`     | [`DateTime`](/composer/builder/api/schema/scalar/DateTime.mdx)           | Filter for objects where createdAt occurs on or before the specified value.                      |
| `updatedAt_Gte`     | [`DateTime`](/composer/builder/api/schema/scalar/DateTime.mdx)           | Filter for objects where updatedAt occurs on or after the specified value.                       |

| `externalId`        | [`String`](/composer/builder/api/schema/scalar/String.mdx)               | Filter for objects where externalId matches the specified value.                                 |

## Return fields ([UserConnection!](/composer/builder/api/schema/object/UserConnection.mdx))

###### <b>edges</b> ([[UserEdge]!](/composer/builder/api/schema/object/UserEdge.mdx))

###### <b>nodes</b> ([[User]!](/composer/builder/api/schema/object/User.mdx))

###### <b>pageInfo</b> ([PageInfo!](/composer/builder/api/schema/object/PageInfo.mdx))

###### <b>totalCount</b> ([Int!](/composer/builder/api/schema/scalar/Int.mdx))
