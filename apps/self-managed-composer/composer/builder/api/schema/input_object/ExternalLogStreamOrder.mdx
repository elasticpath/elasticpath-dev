Allows specifying which field and direction to order by.

## Input fields

| Argument    | Type                                                                              | Description                |
| ----------- | --------------------------------------------------------------------------------- | -------------------------- |
| `direction` | [`OrderDirection!`](/composer/builder/api/schema/enum/OrderDirection.mdx)                           | The direction to order by. |
| `field`     | [`ExternalLogStreamOrderField!`](/composer/builder/api/schema/enum/ExternalLogStreamOrderField.mdx) | The field to order by.     |
