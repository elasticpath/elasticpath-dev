Represents a Theme Color of a given Type and the Variant that it is associated with.

## Input fields

| Argument  | Type                                      | Description                                    |
| --------- | ----------------------------------------- | ---------------------------------------------- |
| `type`    | [`String!`](/composer/builder/api/schema/scalar/String.mdx) | The type of Color.                             |
| `variant` | [`String!`](/composer/builder/api/schema/scalar/String.mdx) | The Theme variant the color is associated with |
| `value`   | [`String!`](/composer/builder/api/schema/scalar/String.mdx) | The value of the color.                        |
