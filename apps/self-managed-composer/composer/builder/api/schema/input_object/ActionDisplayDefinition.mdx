Represents a collection of data that defines how a Component Action is displayed.

## Input fields

| Argument      | Type                                       | Description                                                        |
| ------------- | ------------------------------------------ | ------------------------------------------------------------------ |
| `label`       | [`String!`](/composer/builder/api/schema/scalar/String.mdx)  | The name of the Component.                                         |
| `description` | [`String!`](/composer/builder/api/schema/scalar/String.mdx)  | Additional notes about the Component.                              |
| `category`    | [`String`](/composer/builder/api/schema/scalar/String.mdx)   | The category of the Component.                                     |
| `directions`  | [`String`](/composer/builder/api/schema/scalar/String.mdx)   | Notes which may provide insight on the intended use of the Action. |
| `important`   | [`Boolean`](/composer/builder/api/schema/scalar/Boolean.mdx) | Specifies whether the Action is important and/or commonly used.    |
