Represents an expression that is used to reference Configuration
Variables and results from previous steps.

## Input fields

| Argument | Type                                      | Description                  |
| -------- | ----------------------------------------- | ---------------------------- |
| `name`   | [`String!`](/composer/builder/api/schema/scalar/String.mdx) | The name of the Expression.  |
| `type`   | [`String`](/composer/builder/api/schema/scalar/String.mdx)  | The type of the Expression.  |
| `value`  | [`String`](/composer/builder/api/schema/scalar/String.mdx)  | The value of the Expression. |
| `meta`   | [`String`](/composer/builder/api/schema/scalar/String.mdx)  |                              |
