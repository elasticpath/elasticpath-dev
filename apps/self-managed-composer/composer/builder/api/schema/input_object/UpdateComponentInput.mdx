## Input fields

| Argument           | Type                                       | Description                                                    |
| ------------------ | ------------------------------------------ | -------------------------------------------------------------- |
| `starred`          | [`Boolean`](/composer/builder/api/schema/scalar/Boolean.mdx) | Indicates whether the record is starred by the signed-in User. |
| `id`               | [`ID`](/composer/builder/api/schema/scalar/ID.mdx)           | The ID of the Component to mutate.                             |
| `clientMutationId` | [`String`](/composer/builder/api/schema/scalar/String.mdx)   | A unique identifier for the client performing the mutation.    |
