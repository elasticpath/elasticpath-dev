    Updates an Organizations Theme.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_admin_users].

## Input fields ([UpdateThemeInput!](/composer/builder/api/schema/input_object/UpdateThemeInput.mdx))

| Argument           | Type                                                                     | Description                                                      |
| ------------------ | ------------------------------------------------------------------------ | ---------------------------------------------------------------- |
| `colors`           | [`[ThemeColorInput]`](/composer/builder/api/schema/input_object/ThemeColorInput.mdx)       | A list of inputs that describe the colors used in the theme.     |
| `properties`       | [`[ThemePropertyInput]`](/composer/builder/api/schema/input_object/ThemePropertyInput.mdx) | A list of inputs that describe the properties used in the theme. |
| `id`               | [`ID`](/composer/builder/api/schema/scalar/ID.mdx)                                         | The ID of the Theme to mutate.                                   |
| `clientMutationId` | [`String`](/composer/builder/api/schema/scalar/String.mdx)                                 | A unique identifier for the client performing the mutation.      |
