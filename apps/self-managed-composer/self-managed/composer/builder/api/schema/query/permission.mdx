    Returns the specified Permission object.

Access is permitted when any of the following condition(s) are met: 1. Always allowed.

## Input fields

| Argument | Type                              | Description                      |
| -------- | --------------------------------- | -------------------------------- |
| `id`     | [`ID!`](/self-managed/composer/builder/api/schema/scalar/ID.mdx) | The ID of the Permission object. |

## Return fields ([Permission](/self-managed/composer/builder/api/schema/object/Permission.mdx))

###### <b>allowRemove</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Permission.

###### <b>allowUpdate</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Permission.

###### <b>description</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Description of the Permission.

###### <b>id</b> ([ID!](/self-managed/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>name</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Name of the Permission.

###### <b>objType</b> ([AuthObjectType!](/self-managed/composer/builder/api/schema/object/AuthObjectType.mdx))

The type of object that the Permission is associated with.

###### <b>tag</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

A unique string identity value.
