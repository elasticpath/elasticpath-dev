    Returns the specified AlertTrigger object.

Access is permitted when any of the following condition(s) are met: 1. Always allowed.

## Input fields

| Argument | Type                              | Description                        |
| -------- | --------------------------------- | ---------------------------------- |
| `id`     | [`ID!`](/self-managed/composer/builder/api/schema/scalar/ID.mdx) | The ID of the AlertTrigger object. |

## Return fields ([AlertTrigger](/self-managed/composer/builder/api/schema/object/AlertTrigger.mdx))

###### <b>allowRemove</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the AlertTrigger.

###### <b>allowUpdate</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the AlertTrigger.

###### <b>id</b> ([ID!](/self-managed/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>isInstanceSpecific</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the AlertTrigger is specific to an Instance rather than an InstanceFlowConfig.

###### <b>monitors</b> ([AlertMonitorConnection!](/self-managed/composer/builder/api/schema/object/AlertMonitorConnection.mdx))

The AlertTriggers that are setup to trigger the AlertMonitor.

###### <b>name</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The name of the AlertTrigger.
