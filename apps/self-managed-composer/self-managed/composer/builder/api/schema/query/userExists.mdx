Returns whether a User exists with the specified email address.

## Input fields

| Argument | Type                                      | Description                 |
| -------- | ----------------------------------------- | --------------------------- |
| `email`  | [`String!`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | The email address to check. |

## Return type

###### <b>userExists</b> ([Boolean](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

The `Boolean` scalar type represents `true` or `false`.
