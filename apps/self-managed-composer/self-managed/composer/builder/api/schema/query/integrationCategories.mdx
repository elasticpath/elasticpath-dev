Returns a list of Integration categories.

## Input fields

| Argument        | Type                                       | Description                                                                 |
| --------------- | ------------------------------------------ | --------------------------------------------------------------------------- |
| `onlyLatest`    | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx) | Only include categories that appear on the latest versions of Integrations. |
| `onlyPublished` | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx) | Only include categories that appear on published versions of Integrations.  |

## Return fields ([[IntegrationCategory]!](/self-managed/composer/builder/api/schema/object/IntegrationCategory.mdx))

###### <b>id</b> ([ID!](/self-managed/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>name</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The name of the Integration category.
