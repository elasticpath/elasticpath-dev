    Returns a Relay Connection to a collection of ObjectPermissionGrant objects.

Access is permitted when any of the following condition(s) are met: 1. Always allowed.

## Input fields

| Argument        | Type                                                                                     | Description                                                                                      |
| --------------- | ---------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------ |
| `before`        | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                 | Specifies a cursor for use in combination with `last` to implement backward pagination.          |
| `after`         | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                 | Specifies a cursor for use in combination with `first` to implement forward pagination.          |
| `first`         | [`Int`](/self-managed/composer/builder/api/schema/scalar/Int.mdx)                                                       | A non-negative integer that specifies to return at most `first` edges after the `after` cursor.  |
| `last`          | [`Int`](/self-managed/composer/builder/api/schema/scalar/Int.mdx)                                                       | A non-negative integer that specifies to return at most `last` edges before the `before` cursor. |
| `offset`        | [`Int`](/self-managed/composer/builder/api/schema/scalar/Int.mdx)                                                       |                                                                                                  |
| `user`          | [`ID`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)                                                         | Filter for objects where user matches the specified ID.                                          |
| `obj`           | [`UUID`](/self-managed/composer/builder/api/schema/scalar/UUID.mdx)                                                     | Filter for objects where obj matches the specified value.                                        |
| `permission`    | [`ID`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)                                                         | Filter for objects where permission matches the specified ID.                                    |
| `grantedByRole` | [`ID`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)                                                         | Filter for objects where grantedByRole matches the specified ID.                                 |
| `orderBy`       | [`ObjectPermissionGrantOrder`](/self-managed/composer/builder/api/schema/input_object/ObjectPermissionGrantOrder.mdx)   | DEPRECATED. Prefer using sort_by instead as it allows ordering by many fields.                   |
| `sortBy`        | [`[ObjectPermissionGrantOrder]`](/self-managed/composer/builder/api/schema/input_object/ObjectPermissionGrantOrder.mdx) | Allows specifying many field and direction pairs to sort results by.                             |
