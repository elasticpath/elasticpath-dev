    Disconnect the specified User Level Connection.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_manage_instances]. 2. The specified object is the signed-in User.

## Input fields ([DisconnectUserLevelConnectionInput!](/self-managed/composer/builder/api/schema/input_object/DisconnectUserLevelConnectionInput.mdx))

| Argument           | Type                                     | Description                                                 |
| ------------------ | ---------------------------------------- | ----------------------------------------------------------- |
| `id`               | [`ID`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)         | The ID of the UserLevelConfigVariable to mutate.            |
| `clientMutationId` | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | A unique identifier for the client performing the mutation. |
