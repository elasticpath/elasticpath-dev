    Creates a new OrganizationSigningKey object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_admin_users].

| `publicKey`        | [`String!`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | Public key of the Signing Keypair.                          |
| `clientMutationId` | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)  | A unique identifier for the client performing the mutation. |
