    Creates a new AlertGroup object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_manage_instances].

## Input fields ([CreateAlertGroupInput!](/self-managed/composer/builder/api/schema/input_object/CreateAlertGroupInput.mdx))

| Argument           | Type                                     | Description                                                 |
| ------------------ | ---------------------------------------- | ----------------------------------------------------------- |
| `name`             | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | The name of the AlertGroup                                  |
| `users`            | [`[ID]`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)       | The users in the AlertGroup.                                |
| `webhooks`         | [`[ID]`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)       | The AlertWebhooks in the AlertGroup                         |
| `clientMutationId` | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | A unique identifier for the client performing the mutation. |
