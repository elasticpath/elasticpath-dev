    Updates the specified Organization object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the object: [org_admin_users].

## Input fields ([UpdateOrganizationInput!](/self-managed/composer/builder/api/schema/input_object/UpdateOrganizationInput.mdx))

| Argument           | Type                                       | Description                                                 |
| ------------------ | ------------------------------------------ | ----------------------------------------------------------- |
| `name`             | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)   | The unique name of the Organization.                        |
| `marketplaceName`  | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)   | Display name of the Organization's Marketplace.             |
| `featureFlags`     | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)   |                                                             |
| `labels`           | [`[String]`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | The labels that are associated with the object.             |
| `avatarUrl`        | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)   | The URL for the avatar image.                               |
| `id`               | [`ID`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)           | The ID of the Organization to mutate.                       |
| `clientMutationId` | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)   | A unique identifier for the client performing the mutation. |
