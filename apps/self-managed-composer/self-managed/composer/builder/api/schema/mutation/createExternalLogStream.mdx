    Creates a new ExternalLogStream object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_admin_users].

## Input fields ([CreateExternalLogStreamInput!](/self-managed/composer/builder/api/schema/input_object/CreateExternalLogStreamInput.mdx))

| Argument           | Type                                                                            | Description                                                                                     |
| ------------------ | ------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------- |
| `name`             | [`String!`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                       | Name of the ExternalLogStream.                                                                  |
| `url`              | [`String!`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                       | The URL of the ExternalLogStream.                                                               |
| `payloadTemplate`  | [`String!`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                       | The template that is hydrated and then used as the body of the ExternalLogStream request.       |
| `headers`          | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                        | A JSON string of key/value pairs that will be sent as headers in the ExternalLogStream request. |
| `severityLevels`   | [`[LogSeverityLevelInput]!`](/self-managed/composer/builder/api/schema/input_object/LogSeverityLevelInput.mdx) | The Log severity levels for which Logs should be sent to the ExternalLogStream.                 |
| `clientMutationId` | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                        | A unique identifier for the client performing the mutation.                                     |
