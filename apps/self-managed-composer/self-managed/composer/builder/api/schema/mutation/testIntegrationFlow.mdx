    Initiates execution of an IntegrationFlow for the purposes of testing.

Access is not permitted.

## Input fields ([TestIntegrationFlowInput!](/self-managed/composer/builder/api/schema/input_object/TestIntegrationFlowInput.mdx))

| Argument           | Type                                     | Description                                                                                                     |
| ------------------ | ---------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| `payload`          | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | The payload to send with the POST request that triggers the Integration Flow Test Instance.                     |
| `contentType`      | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | The content type of the payload to send with the POST request that triggers the Integration Flow Test Instance. |
| `headers`          | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | The headers to send with the POST request that triggers the Integration Flow Test Instance.                     |
| `id`               | [`ID`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)         | The ID of the IntegrationFlow to mutate.                                                                        |
| `clientMutationId` | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | A unique identifier for the client performing the mutation.                                                     |
