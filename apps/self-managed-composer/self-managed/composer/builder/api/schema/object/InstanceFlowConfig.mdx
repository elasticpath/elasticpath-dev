Represents the configuration options for a particular IntegrationFlow as it
relates to an Instance.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the InstanceFlowConfig.

###### <b>allowUpdate</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the InstanceFlowConfig.

###### <b>apiKeys</b> ([[String]](/self-managed/composer/builder/api/schema/scalar/String.mdx))

An optional collection of API Keys any of which, when specified, will be required as a header value in all requests to trigger execution of this IntegrationFlow for the associated Instance.

###### <b>executionResults</b> ([InstanceExecutionResultConnection!](/self-managed/composer/builder/api/schema/object/InstanceExecutionResultConnection.mdx))

The specific InstanceFlowConfig for the Instance being executed.

###### <b>flow</b> ([IntegrationFlow!](/self-managed/composer/builder/api/schema/object/IntegrationFlow.mdx))

The IntegrationFlow for which configuration is being specified for the associated Instance.

###### <b>id</b> ([ID!](/self-managed/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>inFailedState</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the latest execution of this InstanceFlowConfig resulted in a failure.

###### <b>instance</b> ([Instance!](/self-managed/composer/builder/api/schema/object/Instance.mdx))

The configuration for the IntegrationFlow associated with the Instance.

###### <b>lastExecutedAt</b> ([DateTime](/self-managed/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the InstanceFlowConfig was most recently executed.

###### <b>logs</b> ([LogConnection!](/self-managed/composer/builder/api/schema/object/LogConnection.mdx))

The IntegrationFlow which created the Log entry.

###### <b>monitors</b> ([AlertMonitorConnection!](/self-managed/composer/builder/api/schema/object/AlertMonitorConnection.mdx))

The IntegrationFlow that is being monitored by the AlertMonitor.

###### <b>testContentType</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Content type of the payload for testing the IntegrationFlow associated with the Instance.

###### <b>testHeaders</b> ([JSONString](/self-managed/composer/builder/api/schema/scalar/JSONString.mdx))

Headers for testing this IntegrationFlow associated with the Instance.

###### <b>testPayload</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Data payload for testing this IntegrationFlow associated with the Instance.

###### <b>webhookUrl</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The URL of the endpoint that triggers execution of the InstanceFlowConfig.
