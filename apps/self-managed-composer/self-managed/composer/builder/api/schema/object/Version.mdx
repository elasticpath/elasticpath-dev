Represents a specific version of an object that implements the Builder versioning protocol.

## Return fields

###### <b>comment</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Additional commentary/description of this Version.

###### <b>id</b> ([ID!](/self-managed/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>isAvailable</b> ([Boolean](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Version is available for use.

###### <b>publishedAt</b> ([DateTime!](/self-managed/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp when the Version was published.

###### <b>publishedBy</b> ([User](/self-managed/composer/builder/api/schema/object/User.mdx))

User that published this Version.

###### <b>versionNumber</b> ([Int](/self-managed/composer/builder/api/schema/scalar/Int.mdx))

The sequential number that corresponds to the Version.
