A Relay edge to a related AlertWebhook object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([AlertWebhook](/self-managed/composer/builder/api/schema/object/AlertWebhook.mdx))

The related object at the end of the edge.
