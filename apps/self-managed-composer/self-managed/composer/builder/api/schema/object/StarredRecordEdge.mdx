A Relay edge to a related StarredRecord object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([StarredRecord](/self-managed/composer/builder/api/schema/object/StarredRecord.mdx))

The related object at the end of the edge.
