Represents a Relay Connection to a collection of CredentialField objects.

## Return fields

###### <b>edges</b> ([[CredentialFieldEdge]!](/self-managed/composer/builder/api/schema/object/CredentialFieldEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[CredentialField]!](/self-managed/composer/builder/api/schema/object/CredentialField.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](/self-managed/composer/builder/api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](/self-managed/composer/builder/api/schema/scalar/Int.mdx))

Total count of nodes available.
