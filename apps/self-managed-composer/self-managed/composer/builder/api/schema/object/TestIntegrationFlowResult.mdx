Result of testing an IntegrationFlow.

## Return fields

###### <b>body</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The HTTP body of the response returned by the InstanceFlow's Trigger.

###### <b>execution</b> ([InstanceExecutionResult](/self-managed/composer/builder/api/schema/object/InstanceExecutionResult.mdx))

The InstanceExecutionResult that specifies the result of testing the IntegrationFlow.

###### <b>flow</b> ([IntegrationFlow](/self-managed/composer/builder/api/schema/object/IntegrationFlow.mdx))

The IntegrationFlow that was tested.

###### <b>headers</b> ([JSONString](/self-managed/composer/builder/api/schema/scalar/JSONString.mdx))

The HTTP headers of the response returned by the InstanceFlow's Trigger.

###### <b>statusCode</b> ([Int](/self-managed/composer/builder/api/schema/scalar/Int.mdx))

The HTTP status code of the response returned by the InstanceFlow's Trigger.
