Represents a user account. A User may belong to an Organization directly or
belong to a Customer, which itself belongs to an Organization.

## Return fields

###### <b>allowChangeRoles</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can change the Role of the User.

###### <b>allowRemove</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the User.

###### <b>allowUpdate</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the User.

###### <b>appAvatarUrl</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The URL for the main avatar image that is displayed in Builder.

###### <b>appName</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The app name displayed in Builder.

###### <b>avatarUrl</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The URL for the avatar image.

###### <b>createdAt</b> ([DateTime!](/self-managed/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was created.

###### <b>customer</b> ([Customer](/self-managed/composer/builder/api/schema/object/Customer.mdx))

The Customer the user belongs to, if any. If this is NULL then Organization will be specified.

###### <b>darkMode</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Designates whether the User has dark mode activated or not.

###### <b>darkModeSyncWithOs</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Designates whether dark mode should be derived from the operating system.

###### <b>dateJoined</b> ([DateTime!](/self-managed/composer/builder/api/schema/scalar/DateTime.mdx))

The date the User was created.

###### <b>email</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The email address associated with the User.

###### <b>externalId</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Allows for mapping an external entity to a Builder record.

###### <b>featureFlags</b> ([JSONString!](/self-managed/composer/builder/api/schema/scalar/JSONString.mdx))

###### <b>id</b> ([ID!](/self-managed/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>marketplaceName</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The name displayed for the Marketplace.

###### <b>name</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The user's preferred name.

###### <b>org</b> ([Organization](/self-managed/composer/builder/api/schema/object/Organization.mdx))

The Organization that the User belongs to, if any. If this is NULL then Customer will be specified.

###### <b>phone</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The preferred contact phone number for the User.

###### <b>role</b> ([Role!](/self-managed/composer/builder/api/schema/object/Role.mdx))

The Role associated with the User which determines its permissions.

###### <b>updatedAt</b> ([DateTime!](/self-managed/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was most recently updated.
