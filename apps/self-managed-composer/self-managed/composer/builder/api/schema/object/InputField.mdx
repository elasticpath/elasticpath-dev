Represents an input field for a Component Action. Defines the basic
properties that must be satisfied by the input data.

## Return fields

###### <b>action</b> ([Action](/self-managed/composer/builder/api/schema/object/Action.mdx))

The Action to which this InputField is associated, if any.

###### <b>allowRemove</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the InputField.

###### <b>allowUpdate</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the InputField.

###### <b>collection</b> ([InputFieldCollection](/self-managed/composer/builder/api/schema/enum/InputFieldCollection.mdx))

Specifies the type of collection to use for storing input values, if applicable.

###### <b>comments</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Additional notes about the InputField.

###### <b>connection</b> ([Connection](/self-managed/composer/builder/api/schema/object/Connection.mdx))

The Connection to which this InputField is associated, if any.

###### <b>default</b> ([JSONOrString](/self-managed/composer/builder/api/schema/scalar/JSONOrString.mdx))

The default value for the InputField.

###### <b>example</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Example valid input for the InputField.

###### <b>id</b> ([ID!](/self-managed/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>key</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

A string which uniquely identifies the InputField in the context of the Action.

###### <b>keyLabel</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Label used for the Keys of a 'keyvaluelist' collection.

###### <b>label</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The name of the InputField.

###### <b>language</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Language to use for the Code Field.

###### <b>model</b> ([JSONString](/self-managed/composer/builder/api/schema/scalar/JSONString.mdx))

Dictates how possible choices are provided for this InputField.

###### <b>onPremiseControlled</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

If true, this input is controlled by the on-premise resource.

###### <b>placeholder</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Placeholder text that will appear in the InputField UI.

###### <b>required</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the InputField is required by the Action.

###### <b>shown</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the InputField is shown in the Designer.

###### <b>type</b> ([InputFieldType!](/self-managed/composer/builder/api/schema/enum/InputFieldType.mdx))

Specifies the type of data the InputField handles.
