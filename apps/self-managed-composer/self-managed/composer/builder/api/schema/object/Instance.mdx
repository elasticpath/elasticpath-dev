Represents an instance of an Integration which has been deployed in the
context of a Customer, to include Config Variable values, Credentials, and
a specific version of an Integration.

## Return fields

###### <b>allowDeploy</b> ([Boolean](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can deploy the Instance.

###### <b>allowRemove</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Instance.

###### <b>allowUpdate</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Instance.

###### <b>allowUpdateConfigVariables</b> ([Boolean](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update config variables for the Instance.

###### <b>configState</b> ([InstanceConfigState](/self-managed/composer/builder/api/schema/enum/InstanceConfigState.mdx))

Describes the state of configuration of this Instance.

###### <b>configVariables</b> ([InstanceConfigVariableConnection!](/self-managed/composer/builder/api/schema/object/InstanceConfigVariableConnection.mdx))

The Instance with which the Config Variable is associated.

###### <b>createdAt</b> ([DateTime!](/self-managed/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was created.

###### <b>customer</b> ([Customer!](/self-managed/composer/builder/api/schema/object/Customer.mdx))

The Customer for which the Instance is deployed.

###### <b>deployedVersion</b> ([Int!](/self-managed/composer/builder/api/schema/scalar/Int.mdx))

The specific version of the Instance that is deployed.

###### <b>description</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Additional notes about the Instance.

###### <b>enabled</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Instance is currently enabled and in an executable state.

###### <b>executionResults</b> ([InstanceExecutionResultConnection!](/self-managed/composer/builder/api/schema/object/InstanceExecutionResultConnection.mdx))

The Instance for which a specific InstanceFlowConfig is being executed.

###### <b>flowConfigs</b> ([InstanceFlowConfigConnection!](/self-managed/composer/builder/api/schema/object/InstanceFlowConfigConnection.mdx))

The configuration for the IntegrationFlow associated with the Instance.

###### <b>id</b> ([ID!](/self-managed/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>inFailedState</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the latest execution of this Instance resulted in a failure.

###### <b>integration</b> ([Integration!](/self-managed/composer/builder/api/schema/object/Integration.mdx))

The Integration that has been deployed for the Instance.

###### <b>isCustomerDeployable</b> ([Boolean](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Instance can be deployed through the Marketplace.

###### <b>isCustomerUpgradeable</b> ([Boolean](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Instance can be upgraded through the Marketplace.

###### <b>labels</b> ([[String!]](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The labels that are associated with the object.

###### <b>lastDeployedAt</b> ([DateTime](/self-managed/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the Instance was most recently deployed.

###### <b>lastExecutedAt</b> ([DateTime](/self-managed/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the Instance was most recently executed.

###### <b>logs</b> ([LogConnection!](/self-managed/composer/builder/api/schema/object/LogConnection.mdx))

The Instance which created the Log entry.

###### <b>logsDisabled</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether to disable the creation of logs during Instance execution.

###### <b>monitors</b> ([AlertMonitorConnection!](/self-managed/composer/builder/api/schema/object/AlertMonitorConnection.mdx))

The Instance that is being monitored by the AlertMonitor.

###### <b>name</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The name of the Instance.

###### <b>needsDeploy</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether a deploy is needed to reflect the newest configuration for this Instance.

###### <b>starred</b> ([Boolean](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Indicates whether the record is starred by the signed-in User.

###### <b>stepResultsDisabled</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether to disable the creation of step results during Instance execution.

###### <b>systemSuspended</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Instance has been suspended by Builder.

###### <b>updatedAt</b> ([DateTime!](/self-managed/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was most recently updated.

###### <b>userLevelConfigVariables</b> ([CustomUserLevelConfigVariableConnection](/self-managed/composer/builder/api/schema/object/CustomUserLevelConfigVariableConnection.mdx))

The User Level Config variables for the requesting User on this Instance.

###### <b>userLevelConfigs</b> ([UserLevelConfigConnection!](/self-managed/composer/builder/api/schema/object/UserLevelConfigConnection.mdx))

The Instance with which the User Level Config is associated.
