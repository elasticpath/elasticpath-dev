Represents snapshots of daily utilization metrics for an Organization.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the OrgDailyUsageMetrics.

###### <b>allowUpdate</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the OrgDailyUsageMetrics.

###### <b>failedExecutionCount</b> ([BigInt!](/self-managed/composer/builder/api/schema/scalar/BigInt.mdx))

The total number of failed Instance executions on the snapshot date.

###### <b>id</b> ([ID!](/self-managed/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>snapshotDate</b> ([Date!](/self-managed/composer/builder/api/schema/scalar/Date.mdx))

The date the utilization metrics snapshot was created.

###### <b>spendMbSecs</b> ([BigInt!](/self-managed/composer/builder/api/schema/scalar/BigInt.mdx))

The total execution spend on the snapshot date in MB-secs.

###### <b>stepCount</b> ([BigInt!](/self-managed/composer/builder/api/schema/scalar/BigInt.mdx))

The total number of steps executed on the snapshot date.

###### <b>successfulExecutionCount</b> ([BigInt!](/self-managed/composer/builder/api/schema/scalar/BigInt.mdx))

The total number of successful Instance executions on the snapshot date.
