Result of testing an InstanceFlowConfig.

## Return fields

###### <b>body</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The HTTP body of the response returned by the InstanceFlowConfig's Trigger.

###### <b>execution</b> ([InstanceExecutionResult](/self-managed/composer/builder/api/schema/object/InstanceExecutionResult.mdx))

The InstanceExecutionResult that specifies the result of testing the InstanceFlowConfig.

###### <b>flowConfig</b> ([InstanceFlowConfig](/self-managed/composer/builder/api/schema/object/InstanceFlowConfig.mdx))

The InstanceFlowConfig that was tested.

###### <b>headers</b> ([JSONString](/self-managed/composer/builder/api/schema/scalar/JSONString.mdx))

The HTTP headers of the response returned by the InstanceFlowConfig's Trigger.

###### <b>statusCode</b> ([Int](/self-managed/composer/builder/api/schema/scalar/Int.mdx))

The HTTP status code of the response returned by the InstanceFlowConfig's Trigger.
