Represents a Connection that is available on a Component.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Connection.

###### <b>allowUpdate</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Connection.

###### <b>comments</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

Additional notes about the Connection.

###### <b>component</b> ([Component!](/self-managed/composer/builder/api/schema/object/Component.mdx))

The Component to which this Connection is associated.

###### <b>default</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies this Connection is the default for the Component.

###### <b>iconUrl</b> ([String](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The URL that specifies where the Connection icon exists.

###### <b>id</b> ([ID!](/self-managed/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>inputs</b> ([InputFieldConnection!](/self-managed/composer/builder/api/schema/object/InputFieldConnection.mdx))

The Connection to which this InputField is associated, if any.

###### <b>key</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

A string which uniquely identifies the Connection in the context of the Action.

###### <b>label</b> ([String!](/self-managed/composer/builder/api/schema/scalar/String.mdx))

The name of the Connection.

###### <b>oauth2Type</b> ([ConnectionOauth2Type](/self-managed/composer/builder/api/schema/enum/ConnectionOauth2Type.mdx))

The OAuth2 flow type, if any, for this Connection.

###### <b>onPremiseAvailable</b> ([Boolean!](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx))

If true, this connection can be used with an on-premise resource.

###### <b>order</b> ([Int](/self-managed/composer/builder/api/schema/scalar/Int.mdx))

Ordering of the Connection.

###### <b>templates</b> ([ConnectionTemplateConnection!](/self-managed/composer/builder/api/schema/object/ConnectionTemplateConnection.mdx))

The Connection from which this template is structured.
