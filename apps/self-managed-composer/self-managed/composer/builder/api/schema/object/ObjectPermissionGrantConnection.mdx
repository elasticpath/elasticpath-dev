Represents a Relay Connection to a collection of ObjectPermissionGrant objects.

## Return fields

###### <b>edges</b> ([[ObjectPermissionGrantEdge]!](/self-managed/composer/builder/api/schema/object/ObjectPermissionGrantEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[ObjectPermissionGrant]!](/self-managed/composer/builder/api/schema/object/ObjectPermissionGrant.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](/self-managed/composer/builder/api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](/self-managed/composer/builder/api/schema/scalar/Int.mdx))

Total count of nodes available.
