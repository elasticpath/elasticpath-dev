## Input fields

| Argument                            | Type                                                                                       | Description                                                                                                                 |
| ----------------------------------- | ------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------- |
| `name`                              | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                   | The name of the Integration.                                                                                                |
| `description`                       | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                   | Additional notes about the Integration.                                                                                     |
| `category`                          | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                   | Specifies the category of the Integration.                                                                                  |
| `customer`                          | [`ID`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)                                                           | The Customer the Integration belongs to, if any. If this is NULL then the Integration belongs to the Organization.          |
| `endpointConfigTestPayload`         | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                   | Data payload for testing the endpoint configuration for this Integration.                                                   |
| `endpointConfigTestContentType`     | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                   | Content type of the payload for testing the endpoint configuration for this Integration.                                    |
| `endpointConfigTestHeaders`         | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                   | A JSON string of key/value pairs that will be sent as headers when testing the endpoint configuration for this Integration. |
| `useAsTemplate`                     | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx)                                                 | Specifies whether the latest published version of this Integration may be used as a template to create new Integrations.    |
| `templateConfiguration`             | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                   | Specifies whether the latest published version of this Integration may be used as a template to create new Integrations.    |
| `allowMultipleMarketplaceInstances` | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx)                                                 | Specifies whether multiple Instances of this Integration may be created from the Marketplace.                               |
| `labels`                            | [`[String]`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                 | The labels that are associated with the object.                                                                             |
| `avatarUrl`                         | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                   | The URL for the avatar image.                                                                                               |
| `testConfigVariables`               | [`[InputInstanceConfigVariable]`](/self-managed/composer/builder/api/schema/input_object/InputInstanceConfigVariable.mdx) | Config Variables that have been specified for the purposes of testing the Integration.                                      |
| `flows`                             | [`[InputIntegrationFlow]`](/self-managed/composer/builder/api/schema/input_object/InputIntegrationFlow.mdx)               | The Integration of which the IntegrationFlow is a part.                                                                     |
| `definition`                        | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                   | The YAML serialized definition of the Integration to import.                                                                |
| `starred`                           | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx)                                                 | Indicates whether the record is starred by the signed-in User.                                                              |
| `addAttachment`                     | [`AttachmentInput`](/self-managed/composer/builder/api/schema/input_object/AttachmentInput.mdx)                           | Adds the specified Attachment to the object.                                                                                |
| `renameAttachment`                  | [`AttachmentRenameInput`](/self-managed/composer/builder/api/schema/input_object/AttachmentRenameInput.mdx)               | Removes the specified Attachment from the object.                                                                           |
| `removeAttachment`                  | [`AttachmentInput`](/self-managed/composer/builder/api/schema/input_object/AttachmentInput.mdx)                           | Renames the specified Attachment on the object.                                                                             |
| `id`                                | [`ID`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)                                                           | The ID of the Integration to mutate.                                                                                        |
| `clientMutationId`                  | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                                   | A unique identifier for the client performing the mutation.                                                                 |
