Allows specifying which field and direction to order by.

## Input fields

| Argument    | Type                                                                    | Description                |
| ----------- | ----------------------------------------------------------------------- | -------------------------- |
| `direction` | [`OrderDirection!`](/self-managed/composer/builder/api/schema/enum/OrderDirection.mdx)                 | The direction to order by. |
| `field`     | [`AlertTriggerOrderField!`](/self-managed/composer/builder/api/schema/enum/AlertTriggerOrderField.mdx) | The field to order by.     |
