## Input fields

| Argument                           | Type                                      | Description                                                                  |
| ---------------------------------- | ----------------------------------------- | ---------------------------------------------------------------------------- |
| `name`                             | [`String!`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | The name of the AlertMonitor.                                                |
| `instance`                         | [`ID`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)          | The Instance that is being monitored by the AlertMonitor.                    |
| `flowConfig`                       | [`ID`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)          | The IntegrationFlow that is being monitored by the AlertMonitor.             |
| `logSeverityLevelCondition`        | [`Int`](/self-managed/composer/builder/api/schema/scalar/Int.mdx)        | The log severity level condition to monitor for relevant AlertTrigger types. |
| `durationSecondsCondition`         | [`Int`](/self-managed/composer/builder/api/schema/scalar/Int.mdx)        | The execution duration condition to monitor for relevant AlertTrigger types. |
| `executionOverdueMinutesCondition` | [`Int`](/self-managed/composer/builder/api/schema/scalar/Int.mdx)        | The execution overdue condition to monitor for relevant AlertTrigger types.  |
| `triggers`                         | [`[ID]`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)        | The AlertTriggers that are setup to trigger the AlertMonitor.                |
| `groups`                           | [`[ID]`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)        | The AlertGroups to notify when the AlertMonitor is triggered.                |
| `users`                            | [`[ID]`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)        | The Users to notify when the AlertMonitor is triggered.                      |
| `webhooks`                         | [`[ID]`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)        | The AlertWebhooks to call when the AlertMonitor is triggered.                |
| `clientMutationId`                 | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)  | A unique identifier for the client performing the mutation.                  |
