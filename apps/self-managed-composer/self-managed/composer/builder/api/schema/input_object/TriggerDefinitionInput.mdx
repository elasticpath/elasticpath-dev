Represents a collection of data that defines a Component Trigger.

## Input fields

| Argument                     | Type                                                                              | Description                                                                    |
| ---------------------------- | --------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| `key`                        | [`String!`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                         | A string which uniquely identifies the Action in the context of the Component. |
| `display`                    | [`ActionDisplayDefinition!`](/self-managed/composer/builder/api/schema/input_object/ActionDisplayDefinition.mdx) | Specifies how the Component Action is displayed.                               |
| `inputs`                     | [`[InputFieldDefinition]!`](/self-managed/composer/builder/api/schema/input_object/InputFieldDefinition.mdx)     | The InputFields supported by the Component Action.                             |
| `authorization`              | [`AuthorizationDefinition`](/self-managed/composer/builder/api/schema/input_object/AuthorizationDefinition.mdx)  | Specifies how the Action handles Authorization.                                |
| `terminateExecution`         | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx)                                        | Specifies whether the Action will terminate Instance execution.                |
| `breakLoop`                  | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx)                                        | Specifies whether an Action will break out of a loop.                          |
| `allowsBranching`            | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx)                                        | Specifies whether the Action will allow Conditional Branching.                 |
| `staticBranchNames`          | [`[String]`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                        | The static branch names associated with an Action.                             |
| `dynamicBranchInput`         | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                          | The input associated with dynamic branching.                                   |
| `examplePayload`             | [`JSONString`](/self-managed/composer/builder/api/schema/scalar/JSONString.mdx)                                  | An example of the returned payload of an Action.                               |
| `synchronousResponseSupport` | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                          | Specifies support for synchronous responses to an Integration webhook request. |
| `scheduleSupport`            | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                          | Specifies support for triggering an Integration on a recurring schedule.       |
| `isCommonTrigger`            | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx)                                        |                                                                                |
| `hasOnInstanceDeploy`        | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx)                                        |                                                                                |
| `hasOnInstanceDelete`        | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx)                                        |                                                                                |
