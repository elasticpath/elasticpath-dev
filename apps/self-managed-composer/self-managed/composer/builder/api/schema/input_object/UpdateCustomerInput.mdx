## Input fields

| Argument                | Type                                                                         | Description                                                                          |
| ----------------------- | ---------------------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| `name`                  | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                     | The name of the Customer, which must be unique within the scope of its Organization. |
| `description`           | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                     | Additional notes about the Customer.                                                 |
| `allowEmbeddedDesigner` | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx)                                   | Specifies whether this Customer can use the Embedded Designer.                       |
| `labels`                | [`[String]`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                   | The labels that are associated with the object.                                      |
| `avatarUrl`             | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                     | The URL for the avatar image.                                                        |
| `externalId`            | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                     | Allows for mapping an external entity to a Builder record.                         |
| `starred`               | [`Boolean`](/self-managed/composer/builder/api/schema/scalar/Boolean.mdx)                                   | Indicates whether the record is starred by the signed-in User.                       |
| `addAttachment`         | [`AttachmentInput`](/self-managed/composer/builder/api/schema/input_object/AttachmentInput.mdx)             | Adds the specified Attachment to the object.                                         |
| `renameAttachment`      | [`AttachmentRenameInput`](/self-managed/composer/builder/api/schema/input_object/AttachmentRenameInput.mdx) | Removes the specified Attachment from the object.                                    |
| `removeAttachment`      | [`AttachmentInput`](/self-managed/composer/builder/api/schema/input_object/AttachmentInput.mdx)             | Renames the specified Attachment on the object.                                      |
| `id`                    | [`ID`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)                                             | The ID of the Customer to mutate.                                                    |
| `clientMutationId`      | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)                                     | A unique identifier for the client performing the mutation.                          |
