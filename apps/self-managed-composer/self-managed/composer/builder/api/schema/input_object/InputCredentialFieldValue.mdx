Represents a specific value of a CredentialField.

## Input fields

| Argument | Type                                      | Description                                   |
| -------- | ----------------------------------------- | --------------------------------------------- |
| `key`    | [`String!`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | The name associated with the CredentialField. |
| `value`  | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx)  | The value of the CredentialField.             |
