## Input fields

| Argument           | Type                                     | Description                                                                    |
| ------------------ | ---------------------------------------- | ------------------------------------------------------------------------------ |
| `email`            | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | The email of the recipient who will complete the OAuth2 authorization request. |
| `message`          | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | The message that will be sent to the recipient of the email.                   |
| `id`               | [`ID`](/self-managed/composer/builder/api/schema/scalar/ID.mdx)         | The ID of the Credential to mutate.                                            |
| `clientMutationId` | [`String`](/self-managed/composer/builder/api/schema/scalar/String.mdx) | A unique identifier for the client performing the mutation.                    |
