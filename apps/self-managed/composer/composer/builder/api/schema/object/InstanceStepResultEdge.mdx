A Relay edge to a related InstanceStepResult object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([InstanceStepResult](/composer/builder/api/schema/object/InstanceStepResult.mdx))

The related object at the end of the edge.
