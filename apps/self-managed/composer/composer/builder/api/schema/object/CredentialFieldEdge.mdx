A Relay edge to a related CredentialField object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([CredentialField](/composer/builder/api/schema/object/CredentialField.mdx))

The related object at the end of the edge.
