Represents the Theme associated with an Organization.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Theme.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Theme.

###### <b>colors</b> ([ThemeColorConnection!](/composer/builder/api/schema/object/ThemeColorConnection.mdx))

The Theme the Color is associated with.

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>properties</b> ([ThemePropertyConnection!](/composer/builder/api/schema/object/ThemePropertyConnection.mdx))

The Theme the Property is associated with.
