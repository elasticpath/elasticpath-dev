Represents a log entry that was created during a specific Execution of an
Instance.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Log.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Log.

###### <b>configVariable</b> ([InstanceConfigVariable](/composer/builder/api/schema/object/InstanceConfigVariable.mdx))

The InstanceConfigVariable which relates to the Log entry.

###### <b>customer</b> ([Customer](/composer/builder/api/schema/object/Customer.mdx))

The specific Customer that is associated with the Log entry.

###### <b>customerName</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

The name of the Customer that is associated with the Log entry.

###### <b>executionResult</b> ([InstanceExecutionResult](/composer/builder/api/schema/object/InstanceExecutionResult.mdx))

The specific InstanceExecutionResult that is associated with the Log entry.

###### <b>flow</b> ([IntegrationFlow](/composer/builder/api/schema/object/IntegrationFlow.mdx))

The specific IntegrationFlow that is associated with the Log entry.

###### <b>flowConfig</b> ([InstanceFlowConfig](/composer/builder/api/schema/object/InstanceFlowConfig.mdx))

The IntegrationFlow which created the Log entry.

###### <b>flowName</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

The name of the IntegrationFlow that is associated with the Log entry.

###### <b>fromPreprocessFlow</b> ([Boolean](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Log was generated as part of the associated Integration's Preprocess Flow.

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>instance</b> ([Instance](/composer/builder/api/schema/object/Instance.mdx))

The Instance which created the Log entry.

###### <b>instanceId</b> ([ID](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the Instance which created the Log entry.

###### <b>instanceName</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

The name of the Instance which created the Log entry.

###### <b>integration</b> ([Integration](/composer/builder/api/schema/object/Integration.mdx))

The specific Integration that is associated with the Log entry.

###### <b>integrationName</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

The name of the Integration that is associated with the Log entry.

###### <b>integrationVersionSequenceId</b> ([UUID](/composer/builder/api/schema/scalar/UUID.mdx))

The identifier for the version sequence of the Integration that is associated with the Log entry.

###### <b>isTestExecution</b> ([Boolean](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the associated Execution was created as part of testing.

###### <b>loopStepIndex</b> ([Int](/composer/builder/api/schema/scalar/Int.mdx))

The iteration index of the containing Loop IntegrationAction at the time this Log entry was generated, if any.

###### <b>loopStepName</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

The name of the IntegrationAction that is the Loop containing the IntegrationAction that generated this Log entry, if any.

###### <b>message</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

The message body of the Log entry.

###### <b>requiredConfigVariableKey</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

The key of the Required Config Variable which relates to the Log entry.

###### <b>severity</b> ([LogSeverityLevel!](/composer/builder/api/schema/enum/LogSeverityLevel.mdx))

The severity level of the Log entry.

###### <b>stepName</b> ([String](/composer/builder/api/schema/scalar/String.mdx))

The name of the IntegrationAction that generated this Log entry.

###### <b>timestamp</b> ([DateTime!](/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the Log was created.

###### <b>userLevelConfigVariable</b> ([UserLevelConfigVariable](/composer/builder/api/schema/object/UserLevelConfigVariable.mdx))

The UserLevelConfigVariable which relates to the Log entry.
