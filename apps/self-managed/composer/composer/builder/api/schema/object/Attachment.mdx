Represents the collection of data that defines an Attachment.

## Return fields

###### <b>name</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

The name of the Attachment.

###### <b>url</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

The URL where the Attachment is located.
