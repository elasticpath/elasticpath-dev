## Input fields

| Argument              | Type                                             | Description                                                                                |
| --------------------- | ------------------------------------------------ | ------------------------------------------------------------------------------------------ |
| `key`                 | [`String!`](/composer/builder/api/schema/scalar/String.mdx)        | The key of the Required Config Var of the Integration for which a value is being provided. |
| `value`               | [`String`](/composer/builder/api/schema/scalar/String.mdx)         | The value to provide for the specified Required Config Var of the Integration.             |
| `values`              | [`JSONString`](/composer/builder/api/schema/scalar/JSONString.mdx) | The values for nested inputs of the specified Required Config Var of the Integration.      |
| `scheduleType`        | [`String`](/composer/builder/api/schema/scalar/String.mdx)         | The schedule type for the specified Required Config Var of the Integration.                |
| `timeZone`            | [`String`](/composer/builder/api/schema/scalar/String.mdx)         | The timezone for the specified Required Config Var of the Integration.                     |
| `onPremiseResourceId` | [`ID`](/composer/builder/api/schema/scalar/ID.mdx)                 |                                                                                            |
