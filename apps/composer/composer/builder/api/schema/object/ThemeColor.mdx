Represents a Color and its various properties used to style the Organization Theme.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the ThemeColor.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the ThemeColor.

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>type</b> ([ThemeColorType!](/composer/builder/api/schema/enum/ThemeColorType.mdx))

The Type of Theme Color.

###### <b>value</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

The Value of the Theme Color.

###### <b>variant</b> ([ThemeColorVariant!](/composer/builder/api/schema/enum/ThemeColorVariant.mdx))

The Theme Variant this Color will be used with.
