A Relay edge to a related ExternalLogStream object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([ExternalLogStream](/composer/builder/api/schema/object/ExternalLogStream.mdx))

The related object at the end of the edge.
