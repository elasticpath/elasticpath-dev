Represents a specific instance of an Event that triggered a specific Alert
Monitor.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the AlertEvent.

###### <b>allowUpdate</b> ([Boolean!](/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the AlertEvent.

###### <b>createdAt</b> ([DateTime!](/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was created.

###### <b>details</b> ([String!](/composer/builder/api/schema/scalar/String.mdx))

Additional information about the event.

###### <b>id</b> ([ID!](/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>monitor</b> ([AlertMonitor!](/composer/builder/api/schema/object/AlertMonitor.mdx))

The AlertMonitor to which the AlertEvent is associated.
