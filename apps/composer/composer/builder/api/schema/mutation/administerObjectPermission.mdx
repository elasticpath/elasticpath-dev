    Administers a Permission to an object for the specified User.

Access is not permitted.

## Input fields ([AdministerObjectPermissionInput!](/composer/builder/api/schema/input_object/AdministerObjectPermissionInput.mdx))

| Argument           | Type                                        | Description                                                     |
| ------------------ | ------------------------------------------- | --------------------------------------------------------------- |
| `grant`            | [`Boolean!`](/composer/builder/api/schema/scalar/Boolean.mdx) | Specifies whether to grant or revoke the specified Permission.  |
| `permission`       | [`ID!`](/composer/builder/api/schema/scalar/ID.mdx)           | The Permission to grant for the specified object.               |
| `object`           | [`ID!`](/composer/builder/api/schema/scalar/ID.mdx)           | The object for which the specified Permission is being granted. |
| `id`               | [`ID`](/composer/builder/api/schema/scalar/ID.mdx)            | The ID of the User to mutate.                                   |
| `clientMutationId` | [`String`](/composer/builder/api/schema/scalar/String.mdx)    | A unique identifier for the client performing the mutation.     |
