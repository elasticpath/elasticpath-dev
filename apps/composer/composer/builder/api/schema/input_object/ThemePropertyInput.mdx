Represents a Property used to style a Theme

## Input fields

| Argument  | Type                                      | Description                                    |
| --------- | ----------------------------------------- | ---------------------------------------------- |
| `type`    | [`String!`](/composer/builder/api/schema/scalar/String.mdx) | The type of Theme Property.                    |
| `value`   | [`String!`](/composer/builder/api/schema/scalar/String.mdx) | The value of the Property.                     |
| `variant` | [`String`](/composer/builder/api/schema/scalar/String.mdx)  | The Theme variant the color is associated with |
