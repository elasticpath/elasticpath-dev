## Input fields

| Argument           | Type                                     | Description                                                                                         |

| `payload`          | [`String`](/composer/builder/api/schema/scalar/String.mdx) | The payload to send with the POST request that triggers the InstanceFlowConfig.                     |
| `contentType`      | [`String`](/composer/builder/api/schema/scalar/String.mdx) | The content type of the payload to send with the POST request that triggers the InstanceFlowConfig. |
| `headers`          | [`String`](/composer/builder/api/schema/scalar/String.mdx) | The headers to send with the POST request that triggers the InstanceFlowConfig.                     |
| `id`               | [`ID`](/composer/builder/api/schema/scalar/ID.mdx)         | The ID of the InstanceFlowConfig to mutate.                                                         |
| `clientMutationId` | [`String`](/composer/builder/api/schema/scalar/String.mdx) | A unique identifier for the client performing the mutation.                                         |
