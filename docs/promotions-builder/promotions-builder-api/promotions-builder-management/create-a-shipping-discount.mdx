## `POST` Create a Shipping Discount

```http
https://useast.api.elasticpath.com/v2/rule-promotions/:promotionID
```
The shipping discount applies to the shipping base price and does not include shipping taxes or additional fees.

## Parameters

### Path parameters

| Name            | Required | Type     | Description                          |
|:----------------|:---------|:---------|:-------------------------------------|
| `promotionID` | Required | `string` | The unique identifier of the promotion. |

### Headers

| Name            | Required | Type     | Description                          |
|:----------------|:---------|:---------|:-------------------------------------|
| `Authorization` | Required | `string` | The Bearer token required to get access to the API. |

## Body

For descriptions about the fields, refer to [Promotions Builder API overview](/docs/promotions-builder/promotions-builder-api/promotions-builder-api-overview).

## Request Example

In the request below, the discount is only applied to shipments with the "regular" `shipping_type`.

```bash
curl -X PUT https://useast.api.elasticpath.com/v2/rule-promotions:promotionID \
     -H "Authorization: Bearer XXXX" \
     -H "Content-Type: application/json" \
     -d $ {
     "data": {
        "type": "rule_promotion",
        "name": "Free regular shipping when cart is $100 or more",
        "description": "Free regular shipping when cart is $100 or more",
        "enabled": true,
        "automatic": false,
        "start": "2024-08-01",
        "end": "2024-12-31",
        "rule_set": {
            "rules": {
                "strategy": "cart_total",
                "operator": "gte",
                "args": [
                    10000
                ]
            },
            "actions": [
                {
                    "strategy": "shipping_discount",
                    "args": [
                        "percent",
                        100
                    ], 
                    "condition":{
                        "strategy": "shipping_type",
                        "operator": "in", 
                        "args": ["regular"]
                    }
                }
            ]
        }
    }
}
```
