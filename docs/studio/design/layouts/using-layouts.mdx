Once you have created your layouts, you need to apply them to your pages.

1. From Studio, select **Content**.
1. Select **Website** to view a list of all your pages.
1. Select **Options** next to the page that you want to add a page layout. The **Page settings** are displayed.
1. In **Layout**, select the layout you want to apply from the list.
1. Select **Save only**.
1. Open the page to check that the layout is working correctly.
1. When you are happy with your page, you can publish your page.
