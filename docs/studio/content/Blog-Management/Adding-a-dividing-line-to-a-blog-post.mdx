---
title: Adding Dividing Line to Blog Post
nav_label: Adding Dividing Line to Blog Post
sidebar_position: 20
---

1. Create a new line by clicking the ``````<Enter>`````` key
2. Once you're on a new, empty line you'll see a greyed icon to the right of your cursor
3. Click the "Plus sign" symbol expand a tray of options:

   Out of this tray, you'll select the second option which is a line icon.

   ![Line Icon](/assets/studio/line-icon.png)

4. This will add the dividing line to the location your cursor is currently on
