## Overview

Helpscout is a customer support email application for managing your conversations with customers.

For extra help, see [adding custom code to Studio](/docs/studio/content/Basic-Page-Editing/embedding-custom-code-in-a-page)

1. Navigate to **Settings** > **Integrations**.
2. Scroll to the custom code section.
3. Click "Add Integration."
4. Name the Integration "Helpscout" and allow the placement to remain "Header."
5. Paste the Beacon code into the code section, and click "Save".
