## Overview

DISQUS is a community forum and comment tool for blogging. Allowing for Studio blogs to accept comments, provide
additional reads, and more.

## Installing DISQUS

1. Gather your code from DISQUS [here](https://disqus.com/admin/install/platforms/universalcode), then edit it following
   along with the video from DISQUS below.
2. Paste the code into
   a [custom section in your blog's footer](/docs/studio/content/Basic-Page-Editing/embedding-custom-code-in-a-page).
