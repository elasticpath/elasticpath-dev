Studio enables you to customize your site's appearance.

If you only need to add custom code to a single page, Studio includes a Custom Code component that can be easily inserted into a page with the **Add Section** button within the page editor.

![Custom code block](/assets/unstack-custom-code-block.png)

To add a custom code block:

1. Click the **Add section** button
2. Navigate to the **Custom Code** category
3. Select the **Custom Code** component

See [Custom Integrations](/docs/studio/Integrations/Adding-HTML-CSS-or-JavaScript-to-all-your-pages).
