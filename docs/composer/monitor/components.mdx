A component is a portion of reusable code that accomplishes a specific task, or connects with an outside service. Several components are strung together to form an integration. Composer offers a wide array of components to accomplish a variety of things. 

In **Components**, you can see a complete list of all the components that Composer provides. 

Select a component for more information about the component, including data sources, connections, and actions.

![epcc component](/assets/epcc_component.png)

In **Integrations**, you can see a complete list of all your integrations. Select an integration, if you want to view or reconfigure the integration.
