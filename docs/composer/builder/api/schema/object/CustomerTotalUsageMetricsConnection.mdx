Represents a Relay Connection to a collection of CustomerTotalUsageMetrics objects.

## Return fields

###### <b>edges</b> ([[CustomerTotalUsageMetricsEdge]!](/docs/composer/builder/api/schema/object/CustomerTotalUsageMetricsEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[CustomerTotalUsageMetrics]!](/docs/composer/builder/api/schema/object/CustomerTotalUsageMetrics.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](/docs/composer/builder/api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](/docs/composer/builder/api/schema/scalar/Int.mdx))

Total count of nodes available.
