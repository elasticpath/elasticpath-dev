Represents configuration for an On-Premise Resource.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the OnPremiseResource.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the OnPremiseResource.

###### <b>customer</b> ([Customer!](/docs/composer/builder/api/schema/object/Customer.mdx))

The Customer associated with this resource.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>name</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of this resource.

###### <b>port</b> ([Int!](/docs/composer/builder/api/schema/scalar/Int.mdx))

The local port on the proxy service assigned to this resource.

###### <b>status</b> ([OnPremiseResourceStatus!](/docs/composer/builder/api/schema/enum/OnPremiseResourceStatus.mdx))

The current status of this On-Premise Resource.
