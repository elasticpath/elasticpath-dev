Represents an object permission, which grants some access to a specific
user relative to a specific object.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Permission.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Permission.

###### <b>description</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

Description of the Permission.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>name</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

Name of the Permission.

###### <b>objType</b> ([AuthObjectType!](/docs/composer/builder/api/schema/object/AuthObjectType.mdx))

The type of object that the Permission is associated with.

###### <b>tag</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

A unique string identity value.
