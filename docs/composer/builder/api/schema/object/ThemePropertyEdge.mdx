A Relay edge to a related ThemeProperty object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([ThemeProperty](/docs/composer/builder/api/schema/object/ThemeProperty.mdx))

The related object at the end of the edge.
