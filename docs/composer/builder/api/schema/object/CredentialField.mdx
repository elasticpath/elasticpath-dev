Represents a specific field on a Credential.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the CredentialField.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the CredentialField.

###### <b>authorizationMethod</b> ([AuthorizationMethod!](/docs/composer/builder/api/schema/object/AuthorizationMethod.mdx))

The AuthorizationMethod that the CredentialField is associated to.

###### <b>comments</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

Additional notes about the CredentialField.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>key</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

A string which uniquely identifies the CredentialField in the context of the AuthorizationMethod.

###### <b>label</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of the CredentialField.

###### <b>placeholder</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

Placeholder text that will appear in the CredentialField UI.

###### <b>required</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the CredentialField requires a value to be valid.

###### <b>type</b> ([CredentialFieldType!](/docs/composer/builder/api/schema/enum/CredentialFieldType.mdx))

Specifies the data type of the value for the CredentialField.
