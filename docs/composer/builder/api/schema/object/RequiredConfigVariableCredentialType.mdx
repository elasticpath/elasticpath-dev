Represents a valid Credential Type for a Required Config Variable.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the RequiredConfigVariableCredentialType.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the RequiredConfigVariableCredentialType.

###### <b>authorizationMethod</b> ([AuthorizationMethod!](/docs/composer/builder/api/schema/object/AuthorizationMethod.mdx))

The Authorization Method that represents a valid Credential type for the Required Config Var.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>requiredConfigVariable</b> ([RequiredConfigVariable!](/docs/composer/builder/api/schema/object/RequiredConfigVariable.mdx))

The Required Config Var for which the Authorization Method is a valid type of Credential.
