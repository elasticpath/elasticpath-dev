Represents an object role, which is just a collection of object permissions
that pertain to a specific object for a specific user.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Role.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Role.

###### <b>description</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

Description of the Role.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>level</b> ([Int!](/docs/composer/builder/api/schema/scalar/Int.mdx))

An integer that specifies the level of privilege with respect to other Roles.

###### <b>name</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of the Role. Must be unique within the context of the AuthObjectType.

###### <b>objType</b> ([AuthObjectType!](/docs/composer/builder/api/schema/object/AuthObjectType.mdx))

The type of object that the Role is associated with.

###### <b>permissions</b> ([PermissionConnection!](/docs/composer/builder/api/schema/object/PermissionConnection.mdx))

List of Permissions that the Role provides.
