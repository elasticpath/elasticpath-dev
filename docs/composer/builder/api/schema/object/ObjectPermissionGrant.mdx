Represents a permission that has been granted to a specified object, either
directly or via a Role.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the ObjectPermissionGrant.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the ObjectPermissionGrant.

###### <b>grantedByRole</b> ([Role](/docs/composer/builder/api/schema/object/Role.mdx))

The Role through which the Permission is granted, if applicable.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>obj</b> ([UUID!](/docs/composer/builder/api/schema/scalar/UUID.mdx))

A reference to the object for which the Permission is granted.

###### <b>permission</b> ([Permission!](/docs/composer/builder/api/schema/object/Permission.mdx))

The Permission being granted.

###### <b>user</b> ([User!](/docs/composer/builder/api/schema/object/User.mdx))

The User for which the Permission is granted.
