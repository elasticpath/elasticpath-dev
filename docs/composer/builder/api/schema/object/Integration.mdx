Represents the collection of information that defines an integration, to
include the sequence of Component Actions, or steps, inputs,
the trigger, and other associated data.

## Return fields

###### <b>actions</b> ([IntegrationActionConnection!](/docs/composer/builder/api/schema/object/IntegrationActionConnection.mdx))

The Integration to which the IntegrationAction is associated via the IntegrationFlow.

###### <b>allowFork</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can fork the Integration.

###### <b>allowManageAttachments</b> ([Boolean](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can manage Attachments related to this record.

###### <b>allowMultipleMarketplaceInstances</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether multiple Instances of this Integration may be created from the Marketplace.

###### <b>allowPublish</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can publish the Integration.

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Integration.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Integration.

###### <b>attachments</b> ([[Attachment]](/docs/composer/builder/api/schema/object/Attachment.mdx))

A JSON list of objects where each object has a key for name and URL that together describe the Attachment.

###### <b>avatarUrl</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

The URL for the avatar image.

###### <b>category</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

Specifies the category of the Integration.

###### <b>components</b> ([ComponentConnection!](/docs/composer/builder/api/schema/object/ComponentConnection.mdx))

Components associated with this Integration

###### <b>configPages</b> ([JSONString!](/docs/composer/builder/api/schema/scalar/JSONString.mdx))

A JSON string that represents deployment configuration pages.

###### <b>createdAt</b> ([DateTime!](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was created.

###### <b>customer</b> ([Customer](/docs/composer/builder/api/schema/object/Customer.mdx))

The Customer the Integration belongs to, if any. If this is NULL then the Integration belongs to the Organization.

###### <b>definition</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

The YAML that is the declarative definition for the Integration. Suitable for using to re-import the Integration.

###### <b>description</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

Additional notes about the Integration.

###### <b>documentation</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

Rich text documentation to accompany the Integration.

###### <b>endpointConfigTestContentType</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

Content type of the payload for testing the endpoint configuration for this Integration.

###### <b>endpointConfigTestHeaders</b> ([JSONString](/docs/composer/builder/api/schema/scalar/JSONString.mdx))

A JSON string of key/value pairs that will be sent as headers when testing the endpoint configuration for this Integration.

###### <b>endpointConfigTestPayload</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

Data payload for testing the endpoint configuration for this Integration.

###### <b>endpointConfigTestUrl</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The URL of the endpoint that allows testing the endpoint configuration of the Integration.

###### <b>endpointType</b> ([IntegrationEndpointType](/docs/composer/builder/api/schema/enum/IntegrationEndpointType.mdx))

Specifies whether endpoint URLs for Instances of this Integration are unique to the flow, unique to the Instance, or if all Instances share a URL.

###### <b>externalVersion</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

Specifies the external version of this Integration, which can be used to provide semantic versioning.

###### <b>flows</b> ([IntegrationFlowConnection!](/docs/composer/builder/api/schema/object/IntegrationFlowConnection.mdx))

The Integration of which the IntegrationFlow is a part.

###### <b>hasOutdatedComponents</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Integration uses outdated Components

###### <b>hasUnpublishedChanges</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Integration definition has changes that have not yet been published.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>instances</b> ([InstanceConnection!](/docs/composer/builder/api/schema/object/InstanceConnection.mdx))

The Integration that has been deployed for the Instance.

###### <b>isCodeNative</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether this Integration is a Code Native Integration.

###### <b>isCustomerDeployable</b> ([Boolean](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Integration can be deployed by the signed-in User.

###### <b>labels</b> ([[String!]](/docs/composer/builder/api/schema/scalar/String.mdx))

The labels that are associated with the object.

###### <b>lastExecutedAt</b> ([DateTime](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which this Integration was most recently executed as part of an Instance.

###### <b>marketplaceConfiguration</b> ([MarketplaceConfiguration!](/docs/composer/builder/api/schema/enum/MarketplaceConfiguration.mdx))

Specifies whether an Integration will be available in the Integration Marketplace and if the Integration is deployable by a Customer User.

###### <b>marketplaceTabConfiguration</b> ([[String!]](/docs/composer/builder/api/schema/scalar/String.mdx))

The Marketplace Tabs available to Customer Users for configuring this Integration.

###### <b>name</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of the Integration.

###### <b>overview</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

Specifies an Overview of the Integration to describe its functionality for use in the Integration Marketplace.

###### <b>parent</b> ([Integration](/docs/composer/builder/api/schema/object/Integration.mdx))

Parent Integration this Integration was forked from, if any

###### <b>preprocessFlowName</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of a Flow in the Integration that will be executed as a preprocessing step prior to any other Flow executions.

###### <b>requiredConfigVariables</b> ([RequiredConfigVariableConnection!](/docs/composer/builder/api/schema/object/RequiredConfigVariableConnection.mdx))

###### <b>starred</b> ([Boolean](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Indicates whether the record is starred by the signed-in User.

###### <b>storeConfiguration</b> ([IntegrationStoreConfiguration](/docs/composer/builder/api/schema/enum/IntegrationStoreConfiguration.mdx))

Specifies whether an Integration will be available in the Integration Store and if the Integration is deployable by a Customer User.

###### <b>systemInstance</b> ([Instance!](/docs/composer/builder/api/schema/object/Instance.mdx))

System Instance backing this Integration.

###### <b>templateConfiguration</b> ([IntegrationTemplateConfiguration!](/docs/composer/builder/api/schema/enum/IntegrationTemplateConfiguration.mdx))

Specifies whether the latest published version of this Integration may be used as a template to create new Integrations.

###### <b>testConfigVariables</b> ([InstanceConfigVariableConnection!](/docs/composer/builder/api/schema/object/InstanceConfigVariableConnection.mdx))

Config Variables that are used for testing during Integration design.

###### <b>updatedAt</b> ([DateTime!](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was most recently updated.

###### <b>useAsTemplate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the latest published version of this Integration may be used as a template to create new Integrations.

###### <b>userLevelConfigured</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether this Integration uses User Level Configs.

###### <b>versionAt</b> ([Integration](/docs/composer/builder/api/schema/object/Integration.mdx))

Object data at specified version

###### <b>versionComment</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

Additional comments about this version.

###### <b>versionCreatedAt</b> ([DateTime](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

Timestamp of the creation of this version.

###### <b>versionCreatedBy</b> ([User](/docs/composer/builder/api/schema/object/User.mdx))

User that created this version.

###### <b>versionIsAvailable</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Indicates if the version is available for use.

###### <b>versionIsLatest</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Marked if this record is the latest version of this sequence.

###### <b>versionNumber</b> ([Int!](/docs/composer/builder/api/schema/scalar/Int.mdx))

Sequential number identifying this version.

###### <b>versionSequence</b> ([IntegrationConnection!](/docs/composer/builder/api/schema/object/IntegrationConnection.mdx))

Sequence of versions of this Integration

###### <b>versionSequenceId</b> ([UUID](/docs/composer/builder/api/schema/scalar/UUID.mdx))

Identifier for this version sequence.

###### <b>versions</b> ([VersionConnection!](/docs/composer/builder/api/schema/object/VersionConnection.mdx))

The Versions of the Integration that are available.
