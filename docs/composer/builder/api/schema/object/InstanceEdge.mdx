A Relay edge to a related Instance object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([Instance](/docs/composer/builder/api/schema/object/Instance.mdx))

The related object at the end of the edge.
