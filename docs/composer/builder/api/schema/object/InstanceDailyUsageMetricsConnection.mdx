Represents a Relay Connection to a collection of InstanceDailyUsageMetrics objects.

## Return fields

###### <b>edges</b> ([[InstanceDailyUsageMetricsEdge]!](/docs/composer/builder/api/schema/object/InstanceDailyUsageMetricsEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[InstanceDailyUsageMetrics]!](/docs/composer/builder/api/schema/object/InstanceDailyUsageMetrics.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](/docs/composer/builder/api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](/docs/composer/builder/api/schema/scalar/Int.mdx))

Total count of nodes available.
