Represents a Webhook that is used for the purposes of alert notification.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the AlertWebhook.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the AlertWebhook.

###### <b>createdAt</b> ([DateTime!](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was created.

###### <b>groups</b> ([AlertGroupConnection!](/docs/composer/builder/api/schema/object/AlertGroupConnection.mdx))

The AlertWebhooks in the AlertGroup

###### <b>headers</b> ([JSONString](/docs/composer/builder/api/schema/scalar/JSONString.mdx))

A JSON string of key/value pairs that will be sent as headers in the Webhook request.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>monitors</b> ([AlertMonitorConnection!](/docs/composer/builder/api/schema/object/AlertMonitorConnection.mdx))

The AlertWebhooks to call when the AlertMonitor is triggered.

###### <b>name</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of the AlertWebhook.

###### <b>payloadTemplate</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The template that is hydrated and then used as the body of the AlertWebhook request.

###### <b>updatedAt</b> ([DateTime!](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was most recently updated.

###### <b>url</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The URL of the AlertWebhook.
