A Relay edge to a related Integration object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([Integration](/docs/composer/builder/api/schema/object/Integration.mdx))

The related object at the end of the edge.
