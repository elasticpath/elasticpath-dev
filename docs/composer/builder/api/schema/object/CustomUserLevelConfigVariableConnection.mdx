Represents a Relay Connection to a collection of User Level Config Variable objects.

## Return fields

###### <b>edges</b> ([[CustomUserLevelConfigVariableEdge]!](/docs/composer/builder/api/schema/object/CustomUserLevelConfigVariableEdge.mdx))

Contains the nodes in this connection.

###### <b>nodes</b> ([[UserLevelConfigVariable]!](/docs/composer/builder/api/schema/object/UserLevelConfigVariable.mdx))

List of nodes in this connection

###### <b>pageInfo</b> ([PageInfo!](/docs/composer/builder/api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](/docs/composer/builder/api/schema/scalar/Int.mdx))

Total count of config variables
