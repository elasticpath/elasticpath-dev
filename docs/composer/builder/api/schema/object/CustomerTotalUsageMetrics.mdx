Represents snapshots of total utilization metrics for a Customer.

## Return fields

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the CustomerTotalUsageMetrics.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the CustomerTotalUsageMetrics.

###### <b>customer</b> ([Customer!](/docs/composer/builder/api/schema/object/Customer.mdx))

The Customer for which utilization metrics are being collected.

###### <b>deployedInstanceCount</b> ([Int!](/docs/composer/builder/api/schema/scalar/Int.mdx))

The total number of Instances that are deployed.

###### <b>deployedUniqueIntegrationCount</b> ([Int!](/docs/composer/builder/api/schema/scalar/Int.mdx))

The total number of unique Integrations that are deployed.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>snapshotTime</b> ([DateTime!](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

The time the utilization metrics snapshot was created.

###### <b>userCount</b> ([Int!](/docs/composer/builder/api/schema/scalar/Int.mdx))

The total number of Users that currently exist.
