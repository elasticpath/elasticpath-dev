A Relay edge to a related Component object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([Component](/docs/composer/builder/api/schema/object/Component.mdx))

The related object at the end of the edge.
