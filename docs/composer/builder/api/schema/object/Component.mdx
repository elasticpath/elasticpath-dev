Represents a package of related functions, or Actions, that can be added to
an Integration.

## Return fields

###### <b>actions</b> ([ActionConnection!](/docs/composer/builder/api/schema/object/ActionConnection.mdx))

The Component to which this Action is associated.

###### <b>allowManageAttachments</b> ([Boolean](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can manage Attachments related to this record.

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Component.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Component.

###### <b>attachments</b> ([[Attachment]](/docs/composer/builder/api/schema/object/Attachment.mdx))

A JSON list of objects where each object has a key for name and URL that together describe the Attachment.

###### <b>category</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

A string that specifies the category of the Component.

###### <b>connections</b> ([ConnectionConnection!](/docs/composer/builder/api/schema/object/ConnectionConnection.mdx))

The Component to which this Connection is associated.

###### <b>customer</b> ([Customer](/docs/composer/builder/api/schema/object/Customer.mdx))

The Customer the Component belongs to, if any. If this is NULL then the Component belongs to the Organization.

###### <b>description</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

Additional notes about the Component.

###### <b>documentationUrl</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

The URL associated with the documentation of a Component.

###### <b>forCodeNativeIntegration</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Component was created inline as part of a Code Native Integration.

###### <b>iconUrl</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

The URL that specifies where the Component icon exists.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>key</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

A string that uniquely identifies the Component.

###### <b>label</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of the Component.

###### <b>labels</b> ([[String!]](/docs/composer/builder/api/schema/scalar/String.mdx))

The labels that are associated with the object.

###### <b>public</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Component is publicly available or whether it's private to the Organization.

###### <b>searchTerms</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

A combination of the Component label, Component description, and every Action label and Action description for the Component to be used for searching.

###### <b>signature</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The hex-encoded SHA1 hash of the uploaded Component package.

###### <b>starred</b> ([Boolean](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Indicates whether the record is starred by the signed-in User.

###### <b>versionAt</b> ([Component](/docs/composer/builder/api/schema/object/Component.mdx))

Object data at specified version

###### <b>versionComment</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

Additional comments about this version.

###### <b>versionCreatedAt</b> ([DateTime](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

Timestamp of the creation of this version.

###### <b>versionCreatedBy</b> ([User](/docs/composer/builder/api/schema/object/User.mdx))

User that created this version.

###### <b>versionIsAvailable</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Indicates if the version is available for use.

###### <b>versionIsLatest</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Marked if this record is the latest version of this sequence.

###### <b>versionNumber</b> ([Int!](/docs/composer/builder/api/schema/scalar/Int.mdx))

Sequential number identifying this version.

###### <b>versionSequence</b> ([ComponentConnection!](/docs/composer/builder/api/schema/object/ComponentConnection.mdx))

Sequence of versions of this Component

###### <b>versionSequenceId</b> ([UUID](/docs/composer/builder/api/schema/scalar/UUID.mdx))

Identifier for this version sequence.

###### <b>versions</b> ([VersionConnection](/docs/composer/builder/api/schema/object/VersionConnection.mdx))

The Versions of the Component that are available.
