Represents the set of results of each step of execution for an Instance.

## Return fields

###### <b>allocatedMemoryMb</b> ([Int](/docs/composer/builder/api/schema/scalar/Int.mdx))

The number of MB of memory allocated by the runtime to execute this Execution.

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the InstanceExecutionResult.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the InstanceExecutionResult.

###### <b>canceledByExecution</b> ([InstanceExecutionResult](/docs/composer/builder/api/schema/object/InstanceExecutionResult.mdx))

The Execution with a matching Unique Request ID that caused this Execution to be canceled.

###### <b>endedAt</b> ([DateTime](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which execution ended.

###### <b>error</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

Any error message that occurred as part of Instance execution.

###### <b>flow</b> ([IntegrationFlow](/docs/composer/builder/api/schema/object/IntegrationFlow.mdx))

The specific IntegrationFlow that is associated with this Execution.

###### <b>flowConfig</b> ([InstanceFlowConfig](/docs/composer/builder/api/schema/object/InstanceFlowConfig.mdx))

The specific InstanceFlowConfig for the Instance being executed.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>instance</b> ([Instance](/docs/composer/builder/api/schema/object/Instance.mdx))

The Instance for which a specific InstanceFlowConfig is being executed.

###### <b>integration</b> ([Integration](/docs/composer/builder/api/schema/object/Integration.mdx))

The specific Integration that is associated with this Execution.

###### <b>invokeType</b> ([InstanceExecutionResultInvokeType](/docs/composer/builder/api/schema/enum/InstanceExecutionResultInvokeType.mdx))

The type of origin that this execution was triggered from.

###### <b>isTestExecution</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether Execution was created as part of testing.

###### <b>logs</b> ([LogConnection!](/docs/composer/builder/api/schema/object/LogConnection.mdx))

The specific InstanceExecutionResult that is associated with the Log entry.

###### <b>maxRetryCount</b> ([Int](/docs/composer/builder/api/schema/scalar/Int.mdx))

The maximum number of times that this Execution may be retried before failing.

###### <b>replayForExecution</b> ([InstanceExecutionResult](/docs/composer/builder/api/schema/object/InstanceExecutionResult.mdx))

The Execution for which this Execution is a replay.

###### <b>replays</b> ([InstanceExecutionResultConnection!](/docs/composer/builder/api/schema/object/InstanceExecutionResultConnection.mdx))

The Execution for which this Execution is a replay.

###### <b>requestPayloadMetadataUrl</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The presigned URL to fetch metadata of the request payload that was sent to invoke Instance execution.

###### <b>requestPayloadUrl</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The presigned URL to download the request payload that was sent to invoke Instance execution.

###### <b>responsePayloadMetadataUrl</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The presigned URL to fetch metadata of the response payload that was received from the Instance execution.

###### <b>responsePayloadUrl</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The presigned URL to download the response payload that was received from the Instance execution.

###### <b>retryAttempts</b> ([InstanceExecutionResultConnection!](/docs/composer/builder/api/schema/object/InstanceExecutionResultConnection.mdx))

The Execution for which this Execution is a retry attempt.

###### <b>retryCount</b> ([Int](/docs/composer/builder/api/schema/scalar/Int.mdx))

The number of times that this Execution has been retried.

###### <b>retryForExecution</b> ([InstanceExecutionResult](/docs/composer/builder/api/schema/object/InstanceExecutionResult.mdx))

The Execution for which this Execution is a retry attempt.

###### <b>retryNextAt</b> ([DateTime](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the next scheduled retry will occur.

###### <b>retryUniqueRequestId</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

A Unique Request ID to use for retry request cancellation.

###### <b>spendMbSecs</b> ([Int](/docs/composer/builder/api/schema/scalar/Int.mdx))

The spend for this Execution in MB-secs.

###### <b>startedAt</b> ([DateTime!](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which execution started.

###### <b>status</b> ([ExecutionStatus!](/docs/composer/builder/api/schema/enum/ExecutionStatus.mdx))

The status of the Instance execution.

###### <b>stepCount</b> ([Int](/docs/composer/builder/api/schema/scalar/Int.mdx))

The number of steps in this Execution.

###### <b>stepResults</b> ([InstanceStepResultConnection!](/docs/composer/builder/api/schema/object/InstanceStepResultConnection.mdx))

The InstanceExecutionResult to which the InstanceStepResult is associated.
