A Relay edge to a related Permission object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([Permission](/docs/composer/builder/api/schema/object/Permission.mdx))

The related object at the end of the edge.
