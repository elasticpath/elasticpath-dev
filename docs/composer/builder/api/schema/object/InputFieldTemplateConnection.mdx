Represents a Relay Connection to a collection of InputFieldTemplate objects.

## Return fields

###### <b>edges</b> ([[InputFieldTemplateEdge]!](/docs/composer/builder/api/schema/object/InputFieldTemplateEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[InputFieldTemplate]!](/docs/composer/builder/api/schema/object/InputFieldTemplate.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](/docs/composer/builder/api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](/docs/composer/builder/api/schema/scalar/Int.mdx))

Total count of nodes available.
