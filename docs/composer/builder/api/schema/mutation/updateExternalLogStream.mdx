    Updates the specified ExternalLogStream object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_admin_users].

## Input fields ([UpdateExternalLogStreamInput!](/docs/composer/builder/api/schema/input_object/UpdateExternalLogStreamInput.mdx))

| Argument           | Type                                                                           | Description                                                                                     |
| ------------------ | ------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------- |
| `name`             | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx)                                       | Name of the ExternalLogStream.                                                                  |
| `url`              | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx)                                       | The URL of the ExternalLogStream.                                                               |
| `payloadTemplate`  | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx)                                       | The template that is hydrated and then used as the body of the ExternalLogStream request.       |
| `headers`          | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx)                                       | A JSON string of key/value pairs that will be sent as headers in the ExternalLogStream request. |
| `severityLevels`   | [`[LogSeverityLevelInput]`](/docs/composer/builder/api/schema/input_object/LogSeverityLevelInput.mdx) | The Log severity levels for which Logs should be sent to the ExternalLogStream.                 |
| `id`               | [`ID`](/docs/composer/builder/api/schema/scalar/ID.mdx)                                               | The ID of the ExternalLogStream to mutate.                                                      |
| `clientMutationId` | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx)                                       | A unique identifier for the client performing the mutation.                                     |
