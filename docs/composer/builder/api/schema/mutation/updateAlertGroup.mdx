    Updates the specified AlertGroup object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_manage_instances].

## Input fields ([UpdateAlertGroupInput!](/docs/composer/builder/api/schema/input_object/UpdateAlertGroupInput.mdx))

| Argument           | Type                                     | Description                                                 |
| ------------------ | ---------------------------------------- | ----------------------------------------------------------- |
| `name`             | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx) | The name of the AlertGroup                                  |
| `users`            | [`[ID]`](/docs/composer/builder/api/schema/scalar/ID.mdx)       | The users in the AlertGroup.                                |
| `webhooks`         | [`[ID]`](/docs/composer/builder/api/schema/scalar/ID.mdx)       | The AlertWebhooks in the AlertGroup                         |
| `id`               | [`ID`](/docs/composer/builder/api/schema/scalar/ID.mdx)         | The ID of the AlertGroup to mutate.                         |
| `clientMutationId` | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx) | A unique identifier for the client performing the mutation. |
