    Removes the specified AlertGroup object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_manage_instances].

## Input fields ([DeleteAlertGroupInput!](/docs/composer/builder/api/schema/input_object/DeleteAlertGroupInput.mdx))

| Argument           | Type                                     | Description                                                 |
| ------------------ | ---------------------------------------- | ----------------------------------------------------------- |
| `id`               | [`ID`](/docs/composer/builder/api/schema/scalar/ID.mdx)         | The ID of the AlertGroup to mutate.                         |
| `clientMutationId` | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx) | A unique identifier for the client performing the mutation. |
