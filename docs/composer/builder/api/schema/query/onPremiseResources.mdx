    Returns a Relay Connection to a collection of OnPremiseResource objects.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_admin_customer_permissions, org_manage_customers, org_crud_customers, org_view_customers]. 2. The signed-in User has any of the following permissions for the object's 'customer' attribute: [customer_view, customer_edit, customer_remove, customer_admin_users, customer_view_users, customer_admin_instance_permissions, customer_view_instances, customer_manage_marketplace_integrations, customer_access_marketplace_integrations].

## Input fields

| Argument          | Type                                                                             | Description                                                                                      |
| ----------------- | -------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------ |
| `before`          | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx)                                         | Specifies a cursor for use in combination with `last` to implement backward pagination.          |
| `after`           | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx)                                         | Specifies a cursor for use in combination with `first` to implement forward pagination.          |
| `first`           | [`Int`](/docs/composer/builder/api/schema/scalar/Int.mdx)                                               | A non-negative integer that specifies to return at most `first` edges after the `after` cursor.  |
| `last`            | [`Int`](/docs/composer/builder/api/schema/scalar/Int.mdx)                                               | A non-negative integer that specifies to return at most `last` edges before the `before` cursor. |
| `offset`          | [`Int`](/docs/composer/builder/api/schema/scalar/Int.mdx)                                               |                                                                                                  |
| `name_Icontains`  | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx)                                         | Filter for objects where name contains the specified value (case insensitive).                   |
| `customer`        | [`ID`](/docs/composer/builder/api/schema/scalar/ID.mdx)                                                 | Filter for objects where customer matches the specified ID.                                      |
| `customer_Isnull` | [`Boolean`](/docs/composer/builder/api/schema/scalar/Boolean.mdx)                                       | Filter for objects where customer is NULL.                                                       |
| `status`          | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx)                                         | Filter for objects where status matches the specified value.                                     |
| `orderBy`         | [`OnPremiseResourceOrder`](/docs/composer/builder/api/schema/input_object/OnPremiseResourceOrder.mdx)   | DEPRECATED. Prefer using sort_by instead as it allows ordering by many fields.                   |
| `sortBy`          | [`[OnPremiseResourceOrder]`](/docs/composer/builder/api/schema/input_object/OnPremiseResourceOrder.mdx) | Allows specifying many field and direction pairs to sort results by.                             |

## Return fields ([OnPremiseResourceConnection!](/docs/composer/builder/api/schema/object/OnPremiseResourceConnection.mdx))

###### <b>edges</b> ([[OnPremiseResourceEdge]!](/docs/composer/builder/api/schema/object/OnPremiseResourceEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[OnPremiseResource]!](/docs/composer/builder/api/schema/object/OnPremiseResource.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](/docs/composer/builder/api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](/docs/composer/builder/api/schema/scalar/Int.mdx))

Total count of nodes available.
