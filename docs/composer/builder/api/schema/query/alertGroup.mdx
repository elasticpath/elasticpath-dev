    Returns the specified AlertGroup object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_admin_instance_permissions, org_manage_instances, org_view_instances].

## Input fields

| Argument | Type                              | Description                      |
| -------- | --------------------------------- | -------------------------------- |
| `id`     | [`ID!`](/docs/composer/builder/api/schema/scalar/ID.mdx) | The ID of the AlertGroup object. |

## Return fields ([AlertGroup](/docs/composer/builder/api/schema/object/AlertGroup.mdx))

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the AlertGroup.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the AlertGroup.

###### <b>createdAt</b> ([DateTime!](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was created.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>monitors</b> ([AlertMonitorConnection!](/docs/composer/builder/api/schema/object/AlertMonitorConnection.mdx))

The AlertGroups to notify when the AlertMonitor is triggered.

###### <b>name</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of the AlertGroup

###### <b>updatedAt</b> ([DateTime!](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the object was most recently updated.

###### <b>users</b> ([UserConnection!](/docs/composer/builder/api/schema/object/UserConnection.mdx))

The users in the AlertGroup.

###### <b>webhooks</b> ([AlertWebhookConnection!](/docs/composer/builder/api/schema/object/AlertWebhookConnection.mdx))

The AlertWebhooks in the AlertGroup
