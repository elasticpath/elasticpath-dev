    Returns the specified OrgDailyUsageMetrics object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_view_utilization].

## Input fields

| Argument | Type                              | Description                                |
| -------- | --------------------------------- | ------------------------------------------ |
| `id`     | [`ID!`](/docs/composer/builder/api/schema/scalar/ID.mdx) | The ID of the OrgDailyUsageMetrics object. |

## Return fields ([OrgDailyUsageMetrics](/docs/composer/builder/api/schema/object/OrgDailyUsageMetrics.mdx))

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the OrgDailyUsageMetrics.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the OrgDailyUsageMetrics.

###### <b>failedExecutionCount</b> ([BigInt!](/docs/composer/builder/api/schema/scalar/BigInt.mdx))

The total number of failed Instance executions on the snapshot date.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>snapshotDate</b> ([Date!](/docs/composer/builder/api/schema/scalar/Date.mdx))

The date the utilization metrics snapshot was created.

###### <b>spendMbSecs</b> ([BigInt!](/docs/composer/builder/api/schema/scalar/BigInt.mdx))

The total execution spend on the snapshot date in MB-secs.

###### <b>stepCount</b> ([BigInt!](/docs/composer/builder/api/schema/scalar/BigInt.mdx))

The total number of steps executed on the snapshot date.

###### <b>successfulExecutionCount</b> ([BigInt!](/docs/composer/builder/api/schema/scalar/BigInt.mdx))

The total number of successful Instance executions on the snapshot date.
