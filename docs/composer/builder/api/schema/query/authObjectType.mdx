    Returns the specified AuthObjectType object.

Access is permitted when any of the following condition(s) are met: 1. Always allowed.

## Input fields

| Argument | Type                              | Description                          |
| -------- | --------------------------------- | ------------------------------------ |
| `id`     | [`ID!`](/docs/composer/builder/api/schema/scalar/ID.mdx) | The ID of the AuthObjectType object. |

## Return fields ([AuthObjectType](/docs/composer/builder/api/schema/object/AuthObjectType.mdx))

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the AuthObjectType.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the AuthObjectType.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>name</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

Name of the AuthObjectType.
