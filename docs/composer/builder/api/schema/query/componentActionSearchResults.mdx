Returns a list of Components and Actions that match the search criteria.

## Input fields

| Argument               | Type                                             | Description                                                                                  |
| ---------------------- | ------------------------------------------------ | -------------------------------------------------------------------------------------------- |
| `searchTerms`          | [`String!`](/docs/composer/builder/api/schema/scalar/String.mdx)        | String to use for searching the relevant fields of Components and Actions.                   |
| `componentFilterQuery` | [`JSONString`](/docs/composer/builder/api/schema/scalar/JSONString.mdx) | JSON structure defining a conditional logic expression tree to use for filtering Components. |
| `actionFilterQuery`    | [`JSONString`](/docs/composer/builder/api/schema/scalar/JSONString.mdx) | JSON structure defining a conditional logic expression tree to use for filtering Actions.    |

## Return type

###### <b>componentActionSearchResults</b> ([ComponentActionSearchResult](/docs/composer/builder/api/schema/union/ComponentActionSearchResult.mdx))

Represents search results for a Component or Action.
