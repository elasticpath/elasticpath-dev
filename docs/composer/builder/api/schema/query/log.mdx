    Returns the specified Log object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_admin_instance_permissions, org_manage_instances, org_view_instances]. 2. The signed-in User has any of the following permissions for the object's 'instance_Customer' attribute: [customer_admin_instance_permissions, customer_view_instances, customer_manage_marketplace_integrations]. 3. The signed-in User has any of the following permissions for the object's 'integration_Customer' attribute: [customer_admin_integration_permissions, customer_manage_integrations, customer_view_integrations]. 4. The signed-in User has any of the following permissions for the object's 'instance' attribute: [instance_admin_permissions, instance_view, instance_edit, instance_remove]. 5. The signed-in User has any of the following permissions for any version of the related integration where the object is related to a 'system' instance: [integration_admin_permissions, integration_view, integration_edit, integration_remove].

## Input fields

| Argument | Type                              | Description               |
| -------- | --------------------------------- | ------------------------- |
| `id`     | [`ID!`](/docs/composer/builder/api/schema/scalar/ID.mdx) | The ID of the Log object. |

## Return fields ([Log](/docs/composer/builder/api/schema/object/Log.mdx))

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Log.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Log.

###### <b>configVariable</b> ([InstanceConfigVariable](/docs/composer/builder/api/schema/object/InstanceConfigVariable.mdx))

The InstanceConfigVariable which relates to the Log entry.

###### <b>customer</b> ([Customer](/docs/composer/builder/api/schema/object/Customer.mdx))

The specific Customer that is associated with the Log entry.

###### <b>customerName</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of the Customer that is associated with the Log entry.

###### <b>executionResult</b> ([InstanceExecutionResult](/docs/composer/builder/api/schema/object/InstanceExecutionResult.mdx))

The specific InstanceExecutionResult that is associated with the Log entry.

###### <b>flow</b> ([IntegrationFlow](/docs/composer/builder/api/schema/object/IntegrationFlow.mdx))

The specific IntegrationFlow that is associated with the Log entry.

###### <b>flowConfig</b> ([InstanceFlowConfig](/docs/composer/builder/api/schema/object/InstanceFlowConfig.mdx))

The IntegrationFlow which created the Log entry.

###### <b>flowName</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of the IntegrationFlow that is associated with the Log entry.

###### <b>fromPreprocessFlow</b> ([Boolean](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the Log was generated as part of the associated Integration's Preprocess Flow.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>instance</b> ([Instance](/docs/composer/builder/api/schema/object/Instance.mdx))

The Instance which created the Log entry.

###### <b>instanceId</b> ([ID](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the Instance which created the Log entry.

###### <b>instanceName</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of the Instance which created the Log entry.

###### <b>integration</b> ([Integration](/docs/composer/builder/api/schema/object/Integration.mdx))

The specific Integration that is associated with the Log entry.

###### <b>integrationName</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of the Integration that is associated with the Log entry.

###### <b>integrationVersionSequenceId</b> ([UUID](/docs/composer/builder/api/schema/scalar/UUID.mdx))

The identifier for the version sequence of the Integration that is associated with the Log entry.

###### <b>isTestExecution</b> ([Boolean](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the associated Execution was created as part of testing.

###### <b>loopStepIndex</b> ([Int](/docs/composer/builder/api/schema/scalar/Int.mdx))

The iteration index of the containing Loop IntegrationAction at the time this Log entry was generated, if any.

###### <b>loopStepName</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of the IntegrationAction that is the Loop containing the IntegrationAction that generated this Log entry, if any.

###### <b>message</b> ([String!](/docs/composer/builder/api/schema/scalar/String.mdx))

The message body of the Log entry.

###### <b>requiredConfigVariableKey</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

The key of the Required Config Variable which relates to the Log entry.

###### <b>severity</b> ([LogSeverityLevel!](/docs/composer/builder/api/schema/enum/LogSeverityLevel.mdx))

The severity level of the Log entry.

###### <b>stepName</b> ([String](/docs/composer/builder/api/schema/scalar/String.mdx))

The name of the IntegrationAction that generated this Log entry.

###### <b>timestamp</b> ([DateTime!](/docs/composer/builder/api/schema/scalar/DateTime.mdx))

The timestamp at which the Log was created.

###### <b>userLevelConfigVariable</b> ([UserLevelConfigVariable](/docs/composer/builder/api/schema/object/UserLevelConfigVariable.mdx))

The UserLevelConfigVariable which relates to the Log entry.
