    Returns the Organization Theme of the signed-in User.

Access is permitted when any of the following condition(s) are met: 1. Always allowed.

## Input fields

| Argument | Type | Description |
| -------- | ---- | ----------- |

## Return fields ([Theme](/docs/composer/builder/api/schema/object/Theme.mdx))

###### <b>allowRemove</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Theme.

###### <b>allowUpdate</b> ([Boolean!](/docs/composer/builder/api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Theme.

###### <b>colors</b> ([ThemeColorConnection!](/docs/composer/builder/api/schema/object/ThemeColorConnection.mdx))

The Theme the Color is associated with.

###### <b>id</b> ([ID!](/docs/composer/builder/api/schema/scalar/ID.mdx))

The ID of the object

###### <b>properties</b> ([ThemePropertyConnection!](/docs/composer/builder/api/schema/object/ThemePropertyConnection.mdx))

The Theme the Property is associated with.
