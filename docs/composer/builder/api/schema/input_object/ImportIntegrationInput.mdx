## Input fields

| Argument           | Type                                      | Description                                                                                                        |
| ------------------ | ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| `customer`         | [`ID`](/docs/composer/builder/api/schema/scalar/ID.mdx)          | The Customer the Integration belongs to, if any. If this is NULL then the Integration belongs to the Organization. |
| `definition`       | [`String!`](/docs/composer/builder/api/schema/scalar/String.mdx) | The YAML serialized definition of the Integration to import.                                                       |
| `integrationId`    | [`ID`](/docs/composer/builder/api/schema/scalar/ID.mdx)          | The ID of the Integration being imported.                                                                          |
| `clientMutationId` | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx)  | A unique identifier for the client performing the mutation.                                                        |
