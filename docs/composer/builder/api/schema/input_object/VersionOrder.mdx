Allows specifying which field and direction to order by.

## Input fields

| Argument    | Type                                                                  | Description                |
| ----------- | --------------------------------------------------------------------- | -------------------------- |
| `direction` | [`VersionOrderDirection!`](/docs/composer/builder/api/schema/enum/VersionOrderDirection.mdx) | The direction to order by. |
| `field`     | [`VersionOrderField!`](/docs/composer/builder/api/schema/enum/VersionOrderField.mdx)         | The field to order by.     |
