Represents a collection of data that defines a Component Data Source.

## Input fields

| Argument           | Type                                                                              | Description                                                                                                                                                                                           |
| ------------------ | --------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `key`              | [`String!`](/docs/composer/builder/api/schema/scalar/String.mdx)                                         | A string which uniquely identifies the Data Source in the context of the Component.                                                                                                                   |
| `display`          | [`ActionDisplayDefinition!`](/docs/composer/builder/api/schema/input_object/ActionDisplayDefinition.mdx) | Specifies how the Data Source is displayed.                                                                                                                                                           |
| `inputs`           | [`[InputFieldDefinition]!`](/docs/composer/builder/api/schema/input_object/InputFieldDefinition.mdx)     | The InputFields supported by the Data Source.                                                                                                                                                         |
| `authorization`    | [`AuthorizationDefinition`](/docs/composer/builder/api/schema/input_object/AuthorizationDefinition.mdx)  | Specifies how the Data Source handles Authorization.                                                                                                                                                  |
| `dataSourceType`   | [`String!`](/docs/composer/builder/api/schema/scalar/String.mdx)                                         | The type of the resulting data from the Data Source.                                                                                                                                                  |
| `examplePayload`   | [`JSONString`](/docs/composer/builder/api/schema/scalar/JSONString.mdx)                                  | An example of the returned payload of an Data Source.                                                                                                                                                 |
| `detailDataSource` | [`String`](/docs/composer/builder/api/schema/scalar/String.mdx)                                          | Specifies the key of a Data Source in this Component which can provide additional details about the content for this Data Source, such as example values when selecting particular API object fields. |
