Allows specifying which field and direction to order by.

## Input fields

| Argument    | Type                                                                | Description                |
| ----------- | ------------------------------------------------------------------- | -------------------------- |
| `direction` | [`OrderDirection!`](/docs/composer/builder/api/schema/enum/OrderDirection.mdx)             | The direction to order by. |
| `field`     | [`ConnectionOrderField!`](/docs/composer/builder/api/schema/enum/ConnectionOrderField.mdx) | The field to order by.     |
