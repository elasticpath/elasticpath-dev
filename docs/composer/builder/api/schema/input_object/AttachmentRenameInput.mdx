Represents the collection of data that allows renaming an Attachment.

## Input fields

| Argument  | Type                                      | Description                              |
| --------- | ----------------------------------------- | ---------------------------------------- |
| `name`    | [`String!`](/docs/composer/builder/api/schema/scalar/String.mdx) | The new name for the Attachment.         |
| `oldName` | [`String!`](/docs/composer/builder/api/schema/scalar/String.mdx) | The old name of the Attachment.          |
| `url`     | [`String!`](/docs/composer/builder/api/schema/scalar/String.mdx) | The URL where the Attachment is located. |
