Allows specifying which field and direction to order by.

## Input fields

| Argument    | Type                                                                                      | Description                |
| ----------- | ----------------------------------------------------------------------------------------- | -------------------------- |
| `direction` | [`OrderDirection!`](/docs/composer/builder/api/schema/enum/OrderDirection.mdx)                                   | The direction to order by. |
| `field`     | [`ObjectPermissionGrantOrderField!`](/docs/composer/builder/api/schema/enum/ObjectPermissionGrantOrderField.mdx) | The field to order by.     |
