Represents a single preset input for a ConnectionTemplate

## Input fields

| Argument | Type                                      | Description                                                 |
| -------- | ----------------------------------------- | ----------------------------------------------------------- |
| `key`    | [`String!`](/docs/composer/builder/api/schema/scalar/String.mdx) | The key of an InputField that the value is associated with. |
| `value`  | [`String!`](/docs/composer/builder/api/schema/scalar/String.mdx) | The preset value of the field.                              |
