We support Stripe Elements and recommend that you use the payment method or [source](https://stripe.com/docs/api/sources) with Stripe payments. For more information about generating a payment method on the client-side, see the [Stripe Elements](https://stripe.com/docs/payments/elements) documentation.

Refer to Stripe [Checkout](https://stripe.com/docs/payments/checkout/how-checkout-works) documentation to see how the checkout works.
